块级标签

当做容器，可以容纳内联标签和其它块级标签。主要特点是排他性，在页面布局时独占同一行。标签的宽度（width），高度（height），填充（padding）和边距（margin）都有效。

常见的块级标签

* address - 地址
* blockquote - 块引用
* center - 举中对齐块
* dir - 目录列表
* div - 常用块级标签，也是css layout的主要标签
* dl - 定义列表
* fieldset - form控制组
* form - 交互表单
* h1~6 - 1~6号标题
* hr - 水平分隔线
* isindex - input prompt
* menu - 菜单列表
* noframes - frames可选内容，（对于不支持frame的浏览器显示此区块内容
* noscript - ）可选脚本内容（对于不支持script的浏览器显示此内容）
* ol - 排序表单
* p - 段落
* pre - 格式化文本
* table - 表格
* ul - 非排序列表
* li - 列表选项

内联标签

只能容纳文本或其他的内联标签。主要特点是允许其他标签与其位于一行，标签的宽度（width）和高度（height）不起作用，其宽度就是自身文字或者图片的宽度。填充（padding）和边距（margin）只有左右方向有效，上下方向无效。

常见的内联标签

* a - 锚点，链接
* abbr - 缩写
* acronym - 首字
* b - 粗体(不推荐)
* bdo - bidi override
* big - 大字体
* br - 换行
* cite - 引用
* code - 计算机代码(在引用源码的时候需要)
* dfn - 定义字段
* em - 强调
* font - 字体设定(不推荐)
* i - 斜体
* img - 图片
* input - 输入框
* button - 按钮
* kbd - 定义键盘文本
* label - 表格标签
* q - 短引用
* s - 中划线(不推荐)
* samp - 定义范例计算机代码
* select - 项目选择
* small - 小字体文本
* span - 常用内联容器，定义文本内区块
* strike - 中划线
* strong - 粗体强调
* sub - 下标
* sup - 上标
* textarea - 多行文本输入框
* tt - 电传文本
* u - 下划线
* var - 定义变量

块级标签和内联标签的比较

块级标签 内联标签
独占一行 允许其他标签同处一行
高度，行高及外边距和内边距都可控 高，行高及外边距（上下）和内边距（上下）不可改变，但内外边距（左右）有效。高度可通过line-height设置
宽度默认为100%，除非设定一个宽度 宽度是它的文字或图片的宽度，不可改变
它可以容纳内联标签和其它块标签 只能容纳文本或者其他内联标签


块级标签和内联标签的相互转换

每个标签都有一个display属性，用于确定该属性的类型。比如<div>标签默认display属性值为“block”,成为块级标签，而<span>标签的默认值为“inline”,成为内联标签。

如果要实现内联标签向块级标签转换只需要将display属性设置为block就行了。在<a>标签加上"display:block"，转换成块级标签，宽度和高度值生效，并且可以独占一行。
