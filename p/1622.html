源码nginx-rtmp-module：https://github.com/arut/nginx-rtmp-module

nginx-rtmp-win32：https://github.com/illuspas/nginx-rtmp-win32 集成完成的，内置播流测试工具

nginx：https://github.com/nginx/nginx

nginx-http-flv-module：https://github.com/winshining/nginx-http-flv-module 对 nginx-rtmp-module 的拓展


需要编译ssl，ssl需要perl支持。

nginx-rtmp-module 放到 objs/lib 目录下面。。

auto/configure 命令中加上 --with-select_module --with-http_ssl_module --add-module=objs/lib/nginx-rtmp-module。

之后按nginx的编译方式即可。


编译成功后，需要添加配置文件及新建一些目录，才能正常启动nginx。建议从https://github.com/illuspas/nginx-rtmp-win32下载解压后，直接替换自己编译生成的nginx.exe文件

    configure文件

        2020-01-03     
    auto/configure \
    --with-cc=cl \
    --with-debug \
    --prefix= \
    --conf-path=conf/nginx.conf \
    --pid-path=logs/nginx.pid \
    --http-log-path=logs/access.log \
    --error-log-path=logs/error.log \
    --sbin-path=nginx.exe \
    --http-client-body-temp-path=temp/client_body_temp \
    --http-proxy-temp-path=temp/proxy_temp \
    --http-fastcgi-temp-path=temp/fastcgi_temp \
    --http-scgi-temp-path=temp/scgi_temp \
    --http-uwsgi-temp-path=temp/uwsgi_temp \
    --with-cc-opt=-DFD_SETSIZE=1024 \
    --with-pcre=objs/lib/pcre-8.41 \
    --with-zlib=objs/lib/zlib-1.2.11 \
    --with-openssl=objs/lib/openssl-1.0.2u \
    --with-openssl-opt=no-asm \
    --with-select_module --with-http_ssl_module --add-module=objs/lib/nginx-rtmp-module-master
