介绍

PHP变量以 $ 开头定义，

PHP可以嵌入到HTML页面，在Apache服务器中请求时被解析。按这种表述，和ASP、JSP的编写方式类似。不同的是ASP必须在windows环境下的iis中运行，JSP会被当做Java servlet编译处理。

PHP服务器

PHP集成环境（phpstudy，xampp等）都包含了PHP运行所需要的环境Apache。不用集成环境可以自己配置整合 PHP 和 Apache 。

开发环境（IDE）

任意的文本编辑器多可以，比如 Sublime Text 、Visual Studio Code 等。也有PHP专用的开发环境Apache NetBeans IDE 等。
