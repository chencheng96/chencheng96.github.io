JavaScript,日期对象

日期对象是常用的内置对象，主要用来显示时间和制作日历。

日期对象中的日期分为两部分，日期和时间。日期包括年月日星期，时间包括时分秒毫秒。

创建日期对象

使用系统内置的构造函数来创建日期对象：

new Date();      //创建日期对象，自动获取当前时间

new Date(str);  //str为表达日期的字符串，一般为：月日年小时分钟秒

new Date(ms);  //ms表示距离需要创建的时间和GMT时间1970年1月1日之间相差的毫秒数

new Date(yyyy,mth,dd,hh,mm,ss);  //年份，月份，号，小时数，分钟数，秒数

<script type="text/javascript">
var d1=new Date();   //创建日期对象，自动获取当前时间
var d2=new Date("May 1,2011");   //将字符串转换为日期对象
var d3=new Date(2011,5,1,20,18,30);   //给定时间创建日期对象
var d4=new Date(10000);   //距离1970年1月1日10000毫秒
document.write(d1.toLocaleString()+"</br>");
document.write(d2.toLocaleString()+"</br>");    //2011/5/1 上午12:00:00
document.write(d3.toLocaleString()+"</br>");    //2011/6/1 下午8:18:30
document.write(d4.toLocaleString()+"</br>");    //1970/1/1 上午8:00:10
</script>

格式化日期

将日期格式化为字符串的方法

toDateString()          //把data对象的日期部分转换为字符串

toTimeString()          //把data对象的时间部分转换为字符串

toLocalDateString()   //采用本地时间显示星期，月，日，年

toLocalTimeString()  //采用本地时间显示时，分，秒和地区

toUTCString()          //采用世界时间格式显示完整的UTC日期     一般不用toGMTString()

<script type="text/javascript">
var d1=new Date();   //创建日期对象，自动获取当前时间
document.write(d1.toLocaleString()+"</br>");
document.write(d1.toDateString()+"</br>");
document.write(d1.toTimeString()+"</br>");
document.write(d1.toUTCString()+"</br>");
</script>

获取日期和时间

可以通过get字母开头的方法来分别获取日期和时间

日期对象中获取日期的方法

getFullYear()：取4位数的年份

getUTCFullYear()：返回世界时间格式的4位数年份

getMonth()：返回日期中的月份

getUTCMonth()：返回世界时间格式的月份

getDate()：返回日期中的天数

getUTCDate()：返回世界时间日期中的天数

getDay()：返回星期

getUTCDay()：返回世界时间日期中的星期

日期对象中获取时间的方法

getHours()：返回日期对象中的小时部分

getUTCHours()：世界时间中的小时部分

getMinutes()：分钟

getUTCMinutes()：世界时间中的分钟部分

getSeconds()：秒钟

getUTCSeconds()：世界时间中的秒钟部分

getMilliseconds()：毫秒数

getUTCMilliseconds()：世界时间中的毫秒

getTime()：日期对象中的时间与1970年1月1日相差的毫秒数

getTimezoneOffset()：本地时间与世界时间相差的分钟数

<script type="text/javascript">
var d=new Date();   //创建日期对象，自动获取当前时间
document.write(d.getFullYear()+"</br>");
document.write(d.getUTCFullYear()+"</br>");
document.write(d.getMilliseconds()+"</br>");
document.write(d.getTime()+"</br>");
document.write(d.getTimezoneOffset()+"</br>");
</script>

设置日期和时间

通过set字母开头的方法设置时间与日期

setFullYear(yyyy)：设置4位数的年份

setUTCFullYear(yyyy)：设置世界时间格式的4位数年份

setMonth(mm)：设置日期中的月份

settUTCMonth()mm：设置世界时间格式的月份

setDate(dd)：设置日期中的天数

setUTCDate(dd)：设置世界时间日期中的天数

setHours()：设置日期对象中的小时部分

setUTCHours()：设置时间中的小时部分

setMinutes()：设置分钟

setUTCMinutes()：设置世界时间中的分钟部分

setSeconds()：设置秒钟

setUTCSeconds()：设置世界时间中的秒钟部分

setMilliseconds()：设置毫秒数

setUTCMilliseconds()：设置世界时间中的毫秒
