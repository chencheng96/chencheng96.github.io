File Access,文件存取

对文件系统进行操作，可以取得和处理服务器上所有驱动器的相关信息；取得服务器上所有文件夹和子文件夹的相关信息；可以处理文件夹内的文件。文件存取组件涉及到以下几个对象：FileSystemObject,Drive,File,Folder,TextStream等。

FileSystemObject

用来访问服务器端文件系统的对象，它可以写入或使用文件，文件夹以及目录的路径；也可以获取系统信息。该对象需要配合TextStream对象一起使用，实现文件读写功能。

Set file 对象=Server.CreateObject("Scripting.FileSystemObject")  //创建FileSystemObject

Set text 对象=file 对象.CreateTextFile(filename[,overwrite[,unicode]])    //创建TextStream对象

其中

filename：字符串表达式，用于指定待创建的文件

overwrite：可选参数，值为true或false，指定在文件名已经存在时是否覆盖原文件

Unicode：可选参数，子为true或false，指明是否以Unicode文件格式创建文件

文件写入操作

创建了TextStream对象后可以对该对象进入操作，可以使用的方法：

方法 说明
Write 向文件中写入指定的字符串
Writeln 向文件中写入一行字符串
Writelines 向文件中写入指定行数的字符串
WriteBlankLines 向文件中写入指定行数的空行
创建文件的实例
<%
Set fsol=Server.CreateObject("Scripting.FileSystemObject")       //创建对象实例
Set tsFile=fsol.CreateTextFile(Server.MapPath("WRITE.htm"),True)    //创建一个文本文件并返回TextStream对象
tsFile.WriteLine "<HTML><HEAD>"    //对文件执行写操作
...
tsFile.Close
%>

文件读取操作

从文件中读取信息，需要使用TextStream对象的OpenTextFile方法返回TextStream对象，然后进行文件的读取。创建打开文件对象的语法格式为：

Set text对象=file 对象.OpenTextFile(filename[,iomode[,create[,format]]])

其中

filename：是待打开文件的名称及路径

iomode：用于指定文件输入/输出的模式，为可选参数

create：可选参数，指明filename所指文件不存在时，是否要创建新文件

format：可选参数，指明以何种方式打开文件

文件读取方法：

方法 说明
Read(n) 从文件中读取指定的字符数
Readln 从文件中读出一行
ReadAll 将文件中的所有内容全部读出
Skip(n) 在读取时跳过指定的行数
SkipLine 在读取时跳过一行

TextStream对象的属性

方法 说明
AtEndOfLine 文件指针是否到达了文件行的末尾，值为True/False
AtEndOfStream
 文件指针是否到达了文件的末尾，值为True/False
Column 返回当前指针所在的行号
Line 返回当前指针所在的列号

从文件中读取内容
<%
Set fsol=Server.CreateObject("Scripting.FileSystemObject")
Set tsFile=fsol.OpenTextFile(Server.MapPath("wite.htm"))
While Not tsFile.AtEndOfStream
   Response.Write tsFile.ReadLine
Wend
tsFile.Close
%>

Drive对象

获取驱动的相关信息可以使用FileSystemObject对象，也可以使用Driver对象。可以使用DriveExists,GetDriveName等方法获得驱动器的信息。

file对象.DriveExists(驱动器)

   若该驱动器存在，则该属性的值为true，否则为false

file对象.GetDriveName(path)

   返回该path参数的驱动器的名称，若该路径不存在，返回“”（空串）。

Drive对象要获得驱动器的相关信息，需要使用FileSystemObject对象的GetDrive方法获得Drive对象。

   Set drive 对象=file 对象.GetDrive(驱动器)

Drive对象的属性

属性 描述
AvailableSpace 向用户返回在指定的驱动器或网络共享驱动器上的可用空间容量。
DriveLetter 返回识别本地驱动器或网络共享驱动器的大写字母。
DriveType 返回指定驱动器的类型。
FileSystem 返回指定驱动器所使用的文件系统类型。
FreeSpace 向用户返回在指定的驱动器或网络共享驱动器上的剩余空间容量。
IsReady 如果指定驱动器已就绪，则返回 true。否则返回 false。
Path 返回其后有一个冒号的大写字母，用来指示指定驱动器的路径名。
RootFolder 返回一个文件夹对象，该文件夹代表指定驱动器的根文件夹。
SerialNumber 返回指定驱动器的序列号。
ShareName 返回指定驱动器的网络共享名。
TotalSize 返回指定的驱动器或网络共享驱动器的总容量
VolumeName 设置或者返回指定驱动器的卷标名

创建FileSystemObject对象实例，采取GetDrive方法获取驱动器“C:”的实例，并将驱动器的符号和总容量写入浏览器
<%
Dim fs,d
Set fs=Server.CreateObject("Scripting.FileSystemObject")
Set d=fs.GetDrive("c:")
Response.Write("Drive"&d&":")
Response.Write("Total size in bytes: " &d.TotalSize)
set d=nothing
set fs=nothing
%>
