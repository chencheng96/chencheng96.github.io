css,选择器

每个CSS样式都包含两个基础部分：选择器和声明块。选择器包括type，ID，class等

标签选择器

<style type="text/css">

p{

color: red;

font-size: 12px;

text=align: center;

}

</style>

...

<body>

<p>这是一个p选择器</p>

</body>

该css代码声明了HTML页面中所有的<p>标记，影响该页面所有的<p>标签

设置的标签要和HTML标签同名


类选择器

<style type="text/css">

.center{

text-align: center;

}

</style>

...

<body>

<h1 class="center">h1标签中的文字居中显示<h1>

<p class="center">p标签中的文字居中显示</p>

</body>


类选择器名称必须以一个英文的圆点开头。

类选择器的名称只能使用字母，数字，连字符（-）和下划线（_）

在圆点后，名称始终以字母开头

类名称区分大小写


在使用时直接在标签中进行调用


ID选择器

<style type="text/css">

#banner{

width： 468px;

height: 60px;

background: #cc0000;

}

</style>

...

<body>

<p id="banner">这是网页的banner</p>

</body>

ID选择器名称前用#表示，遵循与类选择器完全相同的命名规则。


如果某一样式在网页上多次使用，应该使用类选择器。

ID选择器用来识别网页上出现一次的部分，如横幅，版权，logo，网页布局，侧边栏，页脚等。


群选择器

<style type="text/css">

h1,p,.red,#banner{

font-size: 16px;

}

</style>

...

<body>


<h1>h1标签中的文字大小为16px<h1>

<p>p标签中的文字大小为16px</p>

<p class="red">p标签中的文字大小为16px</p>


<p id="banner">这是id选择器中的文字大小为16px</p>

</body>

群选择器即创建一个用逗号分隔的选择器列表，利用群体声明的方法将风格相同的css选择器同时声明


通配符选择器

*{

margin: 0;

padding: 0;

}/*将网页中所有标签的内边距设置为0*/

对网页中所有标签初始化，不管有用的还是不用的，过时的还是先进的。


初始化HTML标签，统一设置网页字体大小等


派生选择器

<style type="text/css">

p a{

font-size: 20px;

}

a{

font-size: 20px;

}

</style>

...

<body>

<p>这是p选择器中<a href="#">a标签</a></p>

<a href="#">a标签</a>

</body>


p a{

font-size: 20px;

}

表示<p>标签中的<a>标签字体大小为20像素，其它位置的<a>标签不受影响



选择器的继承与层叠

<style type="text/css">    /*选择器的继承*/

body{

color: red;

}

</style>

...

<body>

<p>这是p选择器中的<strong>strong标签</strong></p>

</body>

左侧<strong>标签中的属性继承了<p>,<body>标签的属性，所以效果应该是<strong>标签中的文字为红色。

一般影响网页元素位置的属性，或者页边距，背景颜色和元素的边框等，都不会被继承。

<style type="text/css">    /*选择器的层叠*/

body{

color: red;

}

p{

color: #blue;

}

strong{

font-size: 20px;

}

</style>

...

<body>

<p>这是p选择器中的<strong>strong标签</strong></p>

</body>

通过继承，多个标签的属性进行叠加，同一出现出现不同属性值时该怎么处理呢？

浏览器会采取“最近的样式优选的原则”，故左侧<strong>标签中的文字应该显示成蓝色。


选择器的优先级

<style type="text/css">    /*选择器的层叠*/

a{

color: red;

}

p a{

color: blue;

}

.info a{

color: green;

}

</style>

...

<body>

<p class="info">我们的邮箱：<a href="#">联系我们</a></p>

</body>

css提供一个公式，根据赋给各种选择器的值来确定样式的权重，最后根据样式的权重值的大小来决定采用哪种样式：

一个标签选择器：1分

一个类选择器：10分

一个ID选择器：100分

一个内部样式选择器：1000分

一个伪类值：10分

最后显示分值大的那个样式。


若有确保某一特定的属性不会被权重值更大的样式覆盖，需在需保护的属性后面插入!important,避免被权重更大的属性覆盖。


a{color: red; !important }      权重：1，保持优先

.info a{color: green; }            权重：10+1=11




伪类

a:link 指网页中正常的，还没访问到的，鼠标也没有移到过或点击过的所有链接
a:visited 已经点击过的链接
a:hover 用户鼠标滑过这个链接时改变链接的展示效果
a:active 当用户点击链接时的效果，作用范围是按下鼠标的那短暂的十亿分之一秒

伪类的一般顺序为：a:link,a:visited,a:hover,a:active，不能混乱
