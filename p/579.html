html,css,文本,格式化

利用css来设置字体，颜色，字号，行距等。更多css属性参考 css样式大全

设置字体（font-family）

<style type="text/css">

h2{

font-family: Verdana,Geneva,sans-serif,宋体,"Times New Roman";

}

p{

font-family:隶属,黑体;

}

</style></head>

<body>

<h2>font-family 属性设置字体</h2>

<p>font-family 属性设置字体</p>

</body>

font-family: Verdana,Geneva,sans-serif,宋体,"Times New Roman";    //浏览器按顺序寻找字体显示，如果所有定义的字体都找不到，则选用计算机系统的默认字体。

Sans-serif字体看起了干净简洁，经常用在英文标题：Arial,Verdana和Formata。

serif字体适合作为段落文字使用：Time New Roman,Times,serif。

中文的宋体相当于serif字体，黑体就相当于Sans-serif字体。


文字颜色(color)

color： #336699;                 //可以缩短为#369

color: rgb(255,200,200);           //(red,green,blue)色调用百分比或者0~255之间的数字表示

color: red;                   //HTML颜色关键字，有17种


修改字号(font-size)

<style type="text/css">

#zh1{

font-size: 20px

}

#zh2{

font-size: 2em

}

#zh3{

font-size: x-large

}

</style></head>

<body>

<p id="zh1">设置字号使用像素</p>

<p id="zh2">设置字号使用em比例参数</p>

<p id="zh3">设置字号使用x-large关键字</p>

</body>

设置字号的三种方法：

  • 固定单位：pt(点),px(像素),in(英寸),cm(厘米),mm(毫米)
  • 关键字：xx-small(极小),x-small(较小),medium(标准大小),large(大),x-large(较大),xx-large(极大)
  • 比例参数：百分比值,em(iem=100%) 设置文字大小为网页上普通文字大小的倍数


斜体和粗体（font_style）

<style type="text/css">

#zh1{

font-style: normal

}

#zh2{

font-style: italic

}

#zh3{

font-style: oblique

}

#zh4{

font-weight: normal

}

#zh5{

font-weight: bold

}

</style></head>

<body>

<p id="zh1">设置正常显示</p>

<p id="zh2">设置斜体1</p>

<p id="zh3">设置斜体2</p>

</body>

HTML中的<em>和<i>标签设置字体为斜体。

<strong>,<b>,<th>和<h1>标签里的文本为粗体。

font-style属性取值

  • normal：正常显示
  • italic：斜体显示文字
  • oblique：比斜体倾斜角度更大

font-weight属性取值

  • normal：正常粗细
  • bold：粗体
  • bolder：加粗体
  • lighter：细体
  • number：9个整百（100~900）,数字越大，字体越粗，但浏览器几乎不支持，基本没有用


格式化字母（text-transform）

<style type="text/css">

#y1{

text-transform: capitalize

}

#y2{

text-transform: uppercase

}

...

</style></head>

<body>

<p id="y1">first big</p>

<p id="y2">all big</p>

</body>

text-transform属性取值

  • none：默认值，无转换发生
  • capitalize：首字母大写
  • uppercase：字母全部大写
  • lowercase：字母全部小写

font-variant属性取值

  • normal：正常
  • small-caps：定义小型的大写字母


文本的修饰（text-decoration）

<style type="text/css">

#y1{

text-decoration: none

}

#y2{

text-decoration: underline

}

...

</style></head>

<body>

<p><a href="#" id="y1">超链接没有下划线</p>

<p id="y2">有下划线</p>

</body>

text-decoration属性取值

  • none：正常显示
  • underline：加下划线
  • overline：上划线
  • line-through：删除线
  • blink：闪烁文本，大多数浏览器无法实现


调整行高（line-height）

<style type="text/css">

#y1{

font-size: 12px; line-height: normal;

}

#y2{

font-size: 12px; line-height: 20px;

}

...

</style></head>

<body>

<p id="y1">设置行高为浏览器默认值</p>

<p id="y2">设置行高为20像素</p>

</body>

line-height的取值

  • normal：浏览器默认值
  • 20px：20像素
  • 150%：字号的150%，若字号为12px，则取值为18px
  • 2：字号的二倍，若字号为12px，则取值为24px


综合设置字体属性（font）

<style type="text/css">

#y1{

font: italic small-caps bold 20px/10px 黑体;

}

</style></head>

<body>

<p id="y1">设置字体的属性</p>

</body>

font属性取值

  • font-style：字体样式
  • font-variant：小型大写字母
  • font-weight：字体加粗
  • font-size：字体大小
  • line-height：行高
  • font-family：字体名称

如不需要其他属性，font一定要写字体大小和字体名称才能起作用，其他属性可不写
