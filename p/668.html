asp,当天时间,系统时间

常用的ASP时间函数 ，以及输出当前的系统时间
1、now() 获取当前系统日期和时间,ASP输出格式：<%=now()%>

2、Year(now()) 获取年份， ASP输出格式：<%=Year(now())%>

3、Month(now()) 获取当前月份，ASP输出格式：<%=Month(now())%>

4、day(now()) 获取当天数，ASP输出格式：<%=day(now())%>

5、hour(now())获取小时数，ASP输出格式：<%=hour(now())%>

6、Minute(now()) 获取分钟数，ASP输出格式：<%=Minute(now())%>

7、Second(now()) 获取秒钟数，ASP输出格式：<%=Second(now())%>

8、date() 获取当前系统日期，ASP输出格式：<%=date()%>

9、time() 获取当前系统时间，ASP输出格式：<%=time()%>

10、weekdayname(weekday(now))获取当前系统星期，ASP输出格式：<%=weekdayname(weekday(now))%>


下面结合一个实例说明该方法的用法，并且介绍一个经常遇到的场景从数据库调用当天的数据进行显示，或者指定时间段的数据。

            一般我们可以用下面的语句实现：SELECT * FROM reweb WHERE (rq >= now()-1) AND (rq <= now()+1) 这样列出来就是最近24小时的。

需要注意的是该方法列出的是距离现在24小时内的数据，并不是当日的数据，当日数据在数据库的存放需要是完整的日期和时间，即now()获取的。


如果想获取当天数据该怎么处理呢，下面给出具体的实现代码：

            SELECT * FROM table  WHERE (time>= date()) AND (time <= date()+1)

time为数据库表中的时间（包括日期和时间）字段，这样就只有当天的数据被选中了。


如果你想选择不同的时间段进行查询，也可以类似的实现。下面给出查找指定日期数据的方法，这样通过改变URL参数的方式来实现

            rs_tody= String(Request.QueryString("a"));
            rs_record.Source = "SELECT *  FROM table  WHERE (Rtime>= date()+ rs_tody ) AND (Rtime <= date()+1+rs_tody) ";

相类似的查询指定时间段等方式，都大同小异，大家可以结合上面实例测试一下。
