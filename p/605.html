JavaScript,数组方法

数组有很多内置的方法,定义数组：var arr=new Array(1,2,3,43,23,12);

toString()方法：将数组转换为字符串

arr.toString();

join()方法：将数组元素连接成字符串

Array.join方法将数组中所有元素都转换成字符串并连接在一起，返回最后生成的字符串

<script type="text/javascript">
 var arr = new Array(3);
arr[0] = "George"
arr[1] = "John"
arr[2] = "Thomas"
document.write(arr.join());
document.write("</br >");
document.write(arr.join("."));
 </script>

push()方法：在数组尾部添加元素

在数组尾部添加元素，可以一次添加一个或多个元素。返回添加元素后数组的长度。

var len=arr.push("java");

pop()方法：删除并返回数组的最后一个元素

删除数组中最后一个元素，并将该元素返回。

var las=arr.pop();

push()和pop()方法允许将数组当做栈来使用。

concat()方法：添加元素并生成新数组

Array.concat()方法在数组的尾部添加元素，返回一个新数组，原数组的元素和长度都不会改变。

<script type="text/javascript">
var arr = new Array(3)
arr[0] = "George"
arr[1] = "John"
arr[2] = "Thomas"
var arr2 = new Array(3)
arr2[0] = "James"
arr2[1] = "Adrew"
arr2[2] = "Martin"
document.write(arr.concat(arr2))
</script>

unshift()方法：在数组头部添加元素

在数组头部添加元素，可以添加一个或多个元素。返回添加元素之后的数组长度。

var len=arr.unshift("java",true);

shift()方法：删除并返回数组的第一个元素

删除数组的第一个元素，并返回该元素的值，使原数组的长度减1，将原数组中其他的所有元素向前移一位。

var firs=arr.shift();

splice()方法：删除,替换或插入数组元素

splice(start,count,value);

start参数是删除，替换或插入数组元素的开始位置，就是数组的下标

count参数是删除，替换的数组的元素的个数，可选

value是要插入数组的值，可选

返回的是一个数组，删除元素时返回已删除的元素，替换元素时，返回的是被替换的元素。

var newarr=arr.splice(2,2); //删除第三第四个元素

slice()方法：返回数组中的一部分元素

返回数组中某一连续部分的元素，不影响原数组中的数据。

slice(start,end);

var arr1=arr.splice(1,3);     //返回数组中第2~4的元素

如果为负数的话，表示从数组的最后一个元素开始计数，如-2为数组的倒数第二个元素。

reverse()方法：颠倒数组中的元素

方法将数组中的元素颠倒顺序。

var arr1=arr.reverse();

sort()方法：对数组元素排序

将数组中的元素排序并返回排序后的数组。

sort();         //没有参数，按字符编码的顺序排序

sort(func);    //参数是一比较函数，这个函数有两个参数，每次比较两个元素先执行这个函数，将两个元素传递给比较函数，如果返回值大于0，交换两个元素，如果返回值小于或等于0，则不进行操作。

<script type="text/javascript">
function sortNumber(a, b)
{
return a - b
}
var arr = new Array(10,5,40,25,1000,1)
document.write(arr + "<br />")
document.write(arr.sort(sortNumber))
</script>

toLocaleString()方法：转换为当地字符串

根据本地计算机的地区设置来转换每一个元素的值。

<script type="text/javascript">
var d=new Date();
var arr=[30,d];
document.write(arr.toLocaleString());
</script>
