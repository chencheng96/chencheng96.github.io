Python的下载安装，py文件的运行

python
打开Python官网，点download中的windows
下载exe后缀的可执行文件，根据自己系统选择32位还是64位。

安装Python 3.5.1
32位和64位的版本安装起来没有区别，双击打开后，第一步要记得勾上add python to Path 选项（不然等会要自己添加），意思是把Python的安装路径添加到系统环境变量的Path变量中。然后选择自定义路径，默认装C盘也可以

安装python时，一定勾选pip install选项，以及添加 path。如果这项漏选了可以再次安装选择，当扩展模块时会用到的（比如数据库模块pyodbc）

下一步默认全选就好了
再下一步，记得勾选第一项，Install for all users， 然后选择自己的目录

安装过程还是比较快的，稍等一会就好

装完之后打开电脑的cmd，验证一下安装是否成功，主要是看环境变量有没有设置好，在cmd中输入python，然后回车，如果出现python的版本号则说明软件装好了。

1. 在cmd中输入Python后就进入了简单的Python解释器，可以运行Python语句。也可以直接用IDLE的Python Shell。

2. python运行py文件：打开cmd将当前目录移动到 .py文件所在目录。>>>python 运行脚本名.py

3. Python的IDLE（在安装Python时需要选中该项）。对需要运行的py文件右键就会出现 Edit with IDLE 。使用IDLE可以直接运行代码并在Python Shell中直接查看结果。提供语法高亮显示以及对代码的缩排支持。调试器可以用于单步跟进代码，识别并纠正bug。
