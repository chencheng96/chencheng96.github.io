浮动

css浮动可以用在任何元素上，取值：left(左浮动),right(右浮动),none(不浮动，默认值)

<style type="text/css">

.m1{
background: #ccc;
width: 300px;
height: 100px;
margin: 0 auto;
}
.m2{
border: 2px solid #333;
background: #fff;
width: 100px;
height: 50px;
margin: 10px;
float: left;

}

.m3{
border: 2px solid #333;
background: #fff;
width: 100px;
height: 50px;
margin: 10px;
float: right;

}

</style>
</head>
<body>
<div class="m1"> 
 <div class="m2">左浮动</div>
<div class="m3">右浮动</div></div>

</body>


清除（clear）

和浮动属性是对立的，去掉某个位置的浮动元素。本质上是迫使它落到浮动元素的下方。

属性的取值

left：左边不允许浮动
right：右边不允许浮动元素
both：两边都不允许
none：默认值，关闭清除属性
