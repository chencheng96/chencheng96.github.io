定义指针 

int *p = NULL;    //定义一个指针变量
int a = 111;
p = &a;    //给指针赋值
*p    //获取p指向地址存储的数据

通过new关键字分配堆内存

使用new关键字分配内存，返回内存地址，赋值给指针。delete关键字将内存归还堆，释放指定的内存 

int *a = new int;  //new 数据类型
delete a;  //释放a指向的内存空间

通过delete释放内存后，指针变量可以继续使用，可以赋值为NULL，以防意外的调用。通过new分配的内存一定要delete防止内存泄漏。delete指针后，没有设置为null,仍指向原来的内存区域的指针称为悬摆指针(dangling)。

使用new在堆中创建对象时。会调用构造函数实例化。 

cat *pcat = new cat;  //cat为定义的类
delete pcat;

使用指针访问成员变量和成员函数 

(*pcat).getName();  //访问成员函数
pcat->getName();

堆中的成员变量（int *age;） 可以在构造函数中对成员函数分配内存，析构函数中释放。

this指针指向函数被调用的对象。this->age = age

const指针

相对于*操作符，const在左对象是常量，const在右指针是常量。 

const int * pone;    //需要将常量赋值给pone
int * const ptwo;    //ptwo需要是常量，初始化赋值后不能再赋值
const int * const three;

const指针与const成员函数

通过声明指向常量对象的指针，指针只能调用常量函数。

引用

引用创建变量的别名 

int &rIntAge = myAge; //之后rIntAge和myAge等价

引用是指向对象的引用，而不是指向类或数据类型的引用。不能使用空引用。

使用引用或指针再函数间进行值传递 

int a=1, b=2;

swap(a,b);  //引用做参数
swap(int &a,int&b){ a=b }

swap(&a,&b);  //指针
swap(int *a,int * b){ *a=*b }

注意：

不能返回指向局部变量引用。

通过const指针解决，引用和指针不想修改值的问题。指针只为传递值到被调用函数中，不想被调用函数生成新的临时变量时使用该方法。
