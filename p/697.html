Key=WaitKey()  //等待一个键的出现，把那个键命名为key
If Key = 32 Then  //如果代号key是13号键的话(13号键就是enter,就是回车键),那么......(这里是要你填的,你想按了回车之后做什么就在这里填)
Else  //否则(意思就是如果你执行了按键精灵后按的不是回车键,就...)
Goto q9 //返回到标记q开始执行(意思就是回到刚开始的标记q再来执行一次)
End If
