JavaScript中的字符串类型，数字类型和布尔类型的数据类型，如果要进行运算时，必须将不同的数据类型转换成相同的数据类型。

转换成字符串

利用每个值都有的toString()方法

null和underfined类型没有toString();

var number=20;   //数字类型

var str=number.toString();      //把数字类型转换成字符串类型

var b=true;   //布尔类型

var arr=["javascript","html","css"];         //数组类型

document.write(str);          //输出字符串20

document.write(arr.toString());       //输出字符串javascript,html,css

document.write(b.toString));         //输出字符串true

使用String()函数

能将任何类型的值转换成字符串。

如果有toString()方法，则调用该方法并返回相应结果；

如果值是underfined，返回"underfined";

如果值是null，返回"null"。

转换成数字类型

使用Number()函数可以把任何数据类型转换成数字类型：


  • 如果是布尔类型（Boolean），true和false分别变为1和0；
  • 如果是null类型，返回0；
  • 如果是undefined，返回NaN；
  • 如果字符串只包含数字，则将其转换为数字，“021”会变成21；
  • 如果字符串是空的，则转换成0；
  • 如果字符串是其他歌手的字符，则转换成NaN。


var s1="3.6";

window.alert(Number(s1));  //返回3.6

var s2="025";

window.alert(Number(s2));  //返回25

另外两个专门用于把字符串转换成数字的函数：parseInt()和parseFloat()。

parseInt()得到整数格式，

var s1=parseInt("236red");  //得到236，后面的字符串过滤了

var s2=parseInt("");  //空字符串，得到NaN

var s3=parseInt("30.8");   //得到30

parseFloat()得到浮点数格式，

var s1=parseInt("236red");   //得到235

var s2=parseInt("30.8");    //得到30.8

转换成布尔(Boolean)类型

使用Boolean()函数，对任何数据类型的值的调用，返回一个true或false。

布尔型 true false
字符串型 任何非空字符串
 空字符串
数字型 任何非零数字 0和NaN
对象 任何对象 NULL

其他常用方法

数字类型转换成字符串类型，可以将其与一个空字符串相连。

var s=256+"";   //将数字类型变成字符串类型

字符串转换成数字类型，可以将其减零，

var s="256"    //字符串类型

var str=s-0;    //转换成数字型

字符串类型或数字类型转换成布尔类型，可以连续使用两次"!"运算符，如：

var s="javascript";   //字符串型

var b1=!!s;   //转换成布尔型

var n=236; //数字型

var b2=!!n;    //转换成布尔型
