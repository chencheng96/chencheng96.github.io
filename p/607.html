JavaScript,对象

对象是客观世界存在的人，事和物体等实体。可以使用“.”作为对象属性的存取操作符

var name=人.name;

人.name="ton";

可以用“.”作为对象行为的存取操作符

人.读书();

在计算机中，用变量表示对象的属性，用函数来表示对象的方法，对象就是具有特定属性和方法的集合。

创建对象

JavaScript中有两种对象，一种是系统内置的对象，另一种是用户自己创建的对象。

使用构造函数创建内置对象

每个内置对象都有一个构造函数，可以使用new操作符来调用构造函数创建对象。

var o=new Object();   //创建一个空对象

var arr=new Array();  //创建一个数组对象

直接创建自定义对象

由用户按照“键—值”对的形式创建对象。

var 对象名称={属性名1: 属性值1,属性名2: 属性值2,...}

直接创建对象时，所有属性放在大括号中，属性之间用,分开，属性名与属性值直接用:分开。

<script type="text/javascript">
var person={name: "Tom",age: 30,height: 175}
document.write("姓名，身高，体重分别为"+person.name+person.age+person.height);
</script>

通过自定义构造函数创建对象

可以批量创建多个相同的对象。

<script type="text/javascript">   //创建有默认值的构造函数
function person(name,age,height){
this.name=name;
this.age=age;
this.height=height;
}
//创建一个对象
var p1=new person("tom",30,173);
document.write("第一个人名字是："+p1.name+"</br>");
document.write("第一个人年龄是："+p1.age+"</br>");
document.write("第一个人身高是："+p1.height+"</br>");
var p2=new person("jarry",35);
document.write("第二个人名字是："+p2.name+"</br>");
document.write("第二个人年龄是："+p2.age+"</br>");
document.write("第二个人身高是："+p2.height+"</br>"); //调用身高的默认属性
</script>
