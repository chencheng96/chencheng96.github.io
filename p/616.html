JavaScript,事件驱动,事件处理

JavaScript和HTML之间的交互是通过用户和浏览器操作页面时引发的事件来处理的。通过事件驱动可以调用JavaScript中的函数或方法，开发人员用这些事件来执行编码进行响应。

在HTML标签属性中调用事件

<script type="text/javascript">
function btclick(){window.alert("单击了按钮，调用函数");}
</script>
<input type="button" name="bt" value="点击我" onclick="btclick()"/>
<input type="button" name="bt" value="点击我" onclick="window.alert('单击了按钮，直接激发')"/>

复制代码在线测试效果

在HTML代码中添加一个按钮，为这个按钮添加一个onclick属性，属性值是一个函数。当用户的鼠标单击了这个按钮，会调用函数。

也可以将激发时间直接写在HTML标签属性中

在JavaScript代码中调用事件

<form><input type="button" name="bt" value="点击我"/></form>
<script type="text/javascript">
function btclick(){window.alert("单击了按钮，调用函数");}
//调用事件
document.forms[0].elements[0].onclick=btclick;
</script>

复制代码在线测试效果

在JavaScript代码中调用对象事件可以增加代码的独立性与模块性。

JavaScript代码要放在表单后面，因为要载入表单事件，然后才激活JavaScript中的代码。

document.forms[0].elements[0].onclick=btclick;     //forms[0]表示HTML中的第一个表单，elements[0]表示表单中的第一个标签元素，这样就指定了input标签

在JavaScript代码中调用事件时，直接输入函数名，不需要后面的括号。

事件的返回值

事件处理程序返回一个值，判断事件处理程序是否正确处理，或者判断是否进行下一步操作。一般返回值是布尔值（true和false）

<script type="text/javascript">
function check(){
var len=document.myform.myname.value.length;
if(len==0){window.alert("没有输入姓名！");return false;}
else{return true;}
}
</script>
<form name="myform" onsubmit="return check()">
姓名：<input type="text" name="myname" value=""/>
<input type="submit" name="" value="提交"/>
</form>

复制代码在线测试效果

return check()就是返回check()函数的值。如果返回值为true，提交表单。如果是false，则不会提交表单。

this操作符

this操作符用来传递参数，表示对象本身。

<script type="text/javascript">
function check1(obj){window.alert(obj.value);}
function check2(str){window.alert(str);}
</script>
<form name="myform">
<input type="button" name="b1" value="这是this操作符返回的值1" onclick="check1(this)"/>
<input type="button" name="b2" value="这是this操作符返回的值2" onclick="check2(this.value)"/>

</form>

复制代码在线测试效果

check1()的参数是一个对象，函数作用是显示对象的value属性值

check2()的参数是一个字符串，作用是显示字符串str的值

浏览器事件

大多数事件由浏览器产生

HTML 4.01/XHTML 1.0规定的事件标准

事件名称 支持标签 说明
blur a,area,label,input,select,textarea,button 对象失去焦点时所激发的事件
change input,select,textarea 值改变时所激发的事件
click 大多数标签 单击鼠标（按下并释放鼠标键）激发的事件
dblclick 大多数标签 双击鼠标时激发的事件
focus a,area,label,input,select,textarea,button
 对象得到焦点时所激发的事件
keydown 大多数标签
 按下键盘键激发的事件
keypress 大多数标签
 按下并释放键盘键时所激发的事件
keyup 大多数标签
 释放键盘键时所激发的事件
load body,frameset 在一个浏览器窗口中加载文档时，或框架集中所有框架文件中的文档激发的事件
mousedown 大多数标签
 按下鼠标（没有释放）时所激发的事件
mousemove 大多数标签
 移动鼠标时激发的事件
mouseover 大多数标签
 鼠标从对象上移开激发的事件
mouseup 大多数标签
 释放鼠标键时所激发的事件
mouseout 大多数标签
 鼠标键从对象上移开时所激发的事件
reset form 重置表单时所激发的事件
select input,textarea 选中文本时所激发的事件
submit form 提交表单时激发的事件
unload body,frameset 卸载浏览器窗口或框架中的文档时所激发的事件
