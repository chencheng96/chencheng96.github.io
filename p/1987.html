正则就是一串有规律的字符串
掌握好正则对于编写shell脚本有很大帮助
各种编程语言中都有正则，原理是一样的

grep sed awk 三剑客

•grep [-cinvABC] 'word' filename
• -c 行数
• -i 不区分大小写
• -n 显示行号
• -v 取反
• -r 遍历所有子目录
• -A 后面跟数字，过滤出符合要求的行以及下面n行
• -B 同上，过滤出符合要求的行以及上面n行
• -C 同上，同时过滤出符合要求的行以及上下各n行
-r 遍历某个目录下的所有文件查找.

[root@docker grep]# grep -cn 'nologin' passwd
21

[root@docker grep]# grep -n 'nologin' passwd
2:bin:x:1:1:bin:/bin:/sbin/nologin
3:daemon:x:2:2:daemon:/sbin:/sbin/nologin
4:adm:x:3:4:adm:/var/adm:/sbin/nologin

[root@docker grep]# grep -c 'nologin' passwd
20
[root@docker grep]# grep -ic 'nologin' passwd
21

[root@docker grep]# grep -v 'nologin' passwd
root:x:0:0:root:/root:/bin/bash
daemon:x:2:2:daemon:/sbin:/sbin/NOLOGIN
sync:x:5:0:sync:/sbin:/bin/sync
shutdown:x:6:0:shutdown:/sbin:/sbin/shutdown
halt:x:7:0:halt:/sbin:/sbin/halt

[root@docker grep]# grep -A2 'root' passwd
root:x:0:0:root:/root:/bin/bash
bin:x:1:1:bin:/bin:/sbin/nologin
daemon:x:2:2:daemon:/sbin:/sbin/NOLOGIN

常用
grep '[0-9]' /etc/passwd

grep -v '[0-9]' /etc/passwd

grep -v '^#' /etc/init

[root@docker grep]# grep -vn '^#' inittab
11:asdasdas

grep '^[^a-zA-Z]' test.txt

^放在[]里代表非，在方括号外代表以什么开头。

[root@docker grep]# grep 'r.o' passwd
root:x:0:0:root:/root:/bin/bash
operator:x:11:0:operator:/root:/sbin/nologin

.代表任意一个字符。

grep 'ro*o' passwd

[root@docker grep]# grep 'o{2}' passwd
{}里的数字表示前面的字符出现的次数{}
root:x:0:0:root:/root:/bin/bash
lp:x:4:7:lp:/var/spool/lpd:/sbin/nologin
mail:x:8:12:mail:/var/spool/mail:/sbin/nologin
operator:x:11:0:operator:/root:/sbin/nologin
postfix:x:89:89::/var/spool/postfix:/sbin/nologin

egrep 'o{2}' passwd 和grep -E 'o{2}' passwd 一样

[root@docker shelllearn]# egrep 'o{2}' testfunc.sh

url:www.runoob.com

grep'o+o' passwd +表示前面的字符出现过1次或多次

•egrep 'oo?' /etc/passwd ？代表0个或1个？前面的字符

•egrep 'root|nologin' /etc/passwd
grep -E 'root|nologin' /etc/passwd | 表示或者

sed ---替换

[root@docker sed]# sed -n '/root/'p passwd
root:x:0:0:root:/root:/bin/bash
operator:x:11:0:operator:/root:/sbin/nologin

-r 参数可以不使用转义了。

[root@docker sed]# sed -n '2'p passwd
bin:x:1:1:bin:/bin:/sbin/nologin

sed -n '25,$'p paaswd 打印25-末行

其他特殊符号和grep一样

-e 参数可以写多个表达式
[root@docker sed]# sed -e '1'p -e '/bus/'p -n passwd
root:x:0:0:root:/root:/bin/bash
systemd-bus-proxy:x:999:997:systemd Bus Proxy:/:/sbin/nologin
dbus:x:81:81:System message bus:/:/sbin/nologin

sed -n '/bus/'Ip passwd 不区分大小写

sed '1,3'd passwd 删掉1-3行，但是实际不删除文件中的内容

sed -i '/user2/'d passwd.txt

sed '1,10s/root/toor/g' passwd.txt 替换

sed -r 's/[a-zA-Z]//g paswd
