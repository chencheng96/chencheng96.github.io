solr开机自启动设置
1、第一种方法 自写bat文件
首先新建一个txt 文件，并更名后缀为bat ,我名字叫solrstart.bat 。在里边填写下边的内容
@echo off
echo 'solr服务启动中'
start "solr" cmd /k "solr start -p 8983"

也可以写成：
@echo off
echo 'solr服务启动中'
solr start -p 8983

然后，把该bat 文件，放到solr-7.3.1\bin 文件夹中。
最后，把这个文件发送到桌面快捷方式 ,把这个文件快捷方式，复制粘贴到开机启动项文件夹中即可，关于不同的windows系统的开机启动项文件夹的位置，可以百度，我的是window 7,可以在左下角开始---->所有程序中找到启动文件夹，打开后粘贴进里边即可。
缺点：这种方式的启动弹出的窗口不能关闭，一关掉，solr就停了，不好。
注意：上边的配置有可能不行，我没有测试过，如果提示找不到路径，那可能就是路径错误，可以多写几步如下：

@echo off
echo 'solr服务启动中'
d:
cd znxdsoft\solr-7.3.1\bin
solr start -p 8983

即先进入到bin 路径里再执行命令即可.。

2、第二种方法 采用nssm添加服务（推荐）
此方法非常简单好用，具体步骤如下：
访问nssm官方下载地址，下载下这个文件夹以后，解压，将里边的如下exe ，拷贝到你的\solr-7.3.1\bin 文件夹中，
进入到\solr-7.3.1\bin 文件夹中，shift +鼠标右击，点击此处打开命令窗口 ,执行如下命令：
nssm install solr
输入上边的内容后，点击install service 即可。
然后在任务管理器 ----服务 中即可看到你注册的服务，点击打开服务 ，找到相应的服务，启动即可，设置为自动 ，可做到开机启动。
这里注意一下nssm 可以设置很多的开机启动服务，非常好用的软件。
