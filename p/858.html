控制结构

条件分支 

if boolean_expression1:
   suite1
elif boolean_expression2:  #可以有0个或多个elif语句
   suit2
...
else:    #冒号（:）不要忘记
   else_suit

若某种情况什么都不用做可以用pass（作为占位符），表示什么都不做。

if...else语句也可以用下面的一句表达式表示出来。如果boolean_expression为True，条件表达式结果为expression1，否则为expression2。 

expression1 if boolean_expression else expression2

循环 

while boolen_expression:
   while_suit
else:    #可选分支。跳出循环，即boolen-expression为False时才执行
   else_suit
while内部若执行continue语句，跳转到循环起始处。若不能正常终止（break语句，return，发生异常），就会跳过所有的else分支。else分支这种情况对于while循环，for...in循环，try...except块都是一样的。 
for expression in iterable:
   for_suit
else:    #执行情况与while类似
   else_suit

异常处理 

try:
   try_suit
except exception_group1 as variable1:
   #至少需要一个except块，但有finally时，可以没有except语句。
   except_suite1
...
except exception_groupN as variableN:
   except_suiteN
else:    #可选，没有异常产生时执行
   else_suite
finally:    #可选，一定会执行
   finally_suite

except Exception:
except:   #方法将捕获所有异常，一般不这样做
  • 如果某异常发生在try块的suite中，那么每个except分支会尝试顺序执行，直到匹配相应异常。如果没有能匹配的异常，程序将终止。
  • 如果没有异常产生，可选的else会执行。
  • 没有异常或异常被处理，finally块会执行。如果异常产生且没有匹配的except块，那么会先执行finally块（确保资源被正确释放），之后在对异常做出反应

自定义异常 

class exceptionName(baseException):pass
#基类应该为Exception类或继承自Exception的类  class MyException(Exception):pass
div=0;
try:
   if div==0:
      raise MyException
      #pass
   5/div
except MyException:
   print("this is MyException")
