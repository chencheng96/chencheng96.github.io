'merge' is not callable 使用keras模块中的merge方法，报错model object is not callable

  • 2020-10-06
  • 机器学习算法分析
  • 公开
简介在进行深度学习时，使用keras模块中的merge方法融合两个模型层时，一直报错model object is not callable。最后才发现，原来在keras最新版本中，没有merge方法了，解决方案就是将keras版本回退到2.1.6及其之前，操作如下： conda install -c conda keras==2.1.2
在进行深度学习时，使用keras模块中的merge方法融合两个模型层时，一直报错model object is not callable。最后才发现，原来在keras最新版本中，没有merge方法了，解决方案就是将keras版本回退到2.1.6及其之前，操作如下：

conda install -c conda keras==2.1.2
