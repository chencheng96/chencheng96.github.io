jdk可以通过华为开源镜像下载：https://repo.huaweicloud.com/java/jdk/

选择合适的安装包下载，这里选择的是tar.gz压缩包，下载后默认位置在/Downloads。

2.移动到安装位置并解压
这里我选择的安装目录为/usr/java/，打开Terminal进行如下操作：

# 建立安装文件夹
$ cd /usr
$ sudo mkdir java

# 赋予目录最高权限，避免一些奇怪的问题
$ sudo chmod 777 java
# 进入/Downloads
$ cd /Downloads

# 移动文件至安装目录（文件名写自己的安装包名称）
$ sudo mv jdk-12.0.1_linux-x64_bin.tar.gz /usr/java

# 解压
$ sudo tar -zxvf jdk-12.0.1_linux-x64_bin.tar.gz

# 查看安装目录名备用
$ ls
我的解压出的文件夹名为jdk-12.0.1，所以记下目录为/usr/java/jdk-12.0.1

3.配置java环境
$ sudo vim /etc/profile
# 如果没有vim也可用regedit
在文件最后加入以下内容：
export JAVA_HOME=/usr/java/jdk-12.0.1
#JAVA_HOME后为前一步最后确定的目录

export CLASSPATH=.:$JAVA_HOME/lib:$JRE_HOME/lib:$CLASSPATH
export PATH=$JAVA_HOME/bin:$JRE_HOME/bin:$PATH
export JRE_HOME=$JAVA_HOME/jre

4.重启
$ reboot
当然如果不太想重启也可以：
$ source /etc/profile
#查看java版本
$ java -version
$ javac -version
