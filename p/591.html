链接

链接的四种状态  a:link中间不能有空格！

a:link{color: blue;} 网页中为访问的链接（初始状态）

a:visited{cllor: red;} 点击链接后状态变成已访问

a:hover{color: #f33;} 鼠标移到链接上时链接的颜色

a:active{color: #900;} 按下鼠标却还没有的那短暂的十亿分之一秒

如果改变顺序，hover和active状态将不起作用


链接的下划线

取消下划线，使用text-decoration属性，把属性设置为none

a: link{text-decoration: none; }    /*取消链接下划线*/

a: hover{text-decoration: underline; }   /*鼠标经过链接时添加下划线*/

下划线是一条1px的实心直线，没办法改变外观，可以用边框来代替，实现更复杂的下划线效果

a{

text-decoration: none;

border-bottom: dashed 1px #000;

}

边框线的粗细，颜色，线型都可修改，文本与下划线的距离可以使用padding属性来调整


按钮形式的链接

<a href="#" class="button">首页</a>

然后利用border，background，padding，color等属性设置不同宽度的边框，线条类型和颜色等产生按钮效果。

加上鼠标移动到链接的效果实现按钮的动态效果。

<style type="text/css">
a{
font-family: 黑体;
font-size: 12px;
text-align: center;
margin: 3px;      /*外边距3像素*/
text-decoration: none;     /*取消下划线*/
}
a:link{               /*设置初始状态时的外观*/
color: #662020;
padding: 4px 10px 4px 10px;
background: #eeddbb;
border-top:1px solid #eee;    /*边框实现阴影效果*/
border-left: 1px solid #eee;
border-bottom: 1px solid #717171;
border-right: 1px solid #717171;
}
a:hover{           /*设置鼠标移动到链接上的效果*/
color: #881818;
padding: 5px 8px 3px 12px;
background: #eecccc;
border-top:1px solid #717171;  /*边框变换，亮暗颜色切换*/
border-left: 1px solid #717171;
border-bottom: 1px solid #eee;
border-right: 1px solid #eee;
}
</style>
</hede>
<body>
<a href="#" class="button">首页</a>
</body>
