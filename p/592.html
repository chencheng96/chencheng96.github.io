表单的美化

CSS中没有用于表单的属性。不同浏览器的效果可能不同

美化文本框

<input type="text" class="text">

使用CSS格式化文本框
<style type="text/css">
.text{
border-left: #fff 1px solid;       /*边框颜色和粗细*/
border-top: #fff 1px solid;
border-right: #fff 1px solid;
border-bottom: #fff 1px solid;
margin: 0px 0px 15px 15px;         /*边距*/
padding-left: 10px;          /*间距*/
background: #ccc;        /*背景色*/
float: left;            /*左浮动*/
font-size: 1em;           /*字体大小*/
line-height: 1.5em;     /*行高*/
height: 20px;        /*文本框高度*/
text-align: left;       /*文字对齐方式*/
}
</style>

点击查看运行效果



美化按钮

使用CSS格式化按钮

<input type="button" class="text"/>

使用CSS格式化按钮
<style type="text/css">
.text{
border:#999 1px solid;      /*边框颜色和粗细*/
line-height: 25px;      /*行高*/
height: 25px;      /*高度*/
width: 100px;
font-size: 16px;       /*字体大小*/
}
</style>

点击查看运行效果
