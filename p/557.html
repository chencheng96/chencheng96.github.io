  • paragraphs : 标题段落正文菜单

    二维数组.
默认值是 [
  ["正文",	"p"],
  ["标题1",	"h1"],
  ["标题2",	"h2"],
  ["标题3",	"h3"],
  ["标题4",	"h4"],
  ["标题5",	"h5"],
  ["标题6",	"h6"],
  ["预定义",	"pre"]
]

  • fontnames : 字体列表

    二维数组.
默认值 [
  ["宋体",            "宋体"],
  ["新宋体",          "新宋体"],
  ["仿宋_GB2312",     "仿宋_GB2312"],
  ["黑体",            "黑体"],
  ["微软雅黑",        "微软雅黑"],
  ["楷体_GB2312",     "楷体_GB2312"],
  ["Arial",           "arial, helvetica, sans-serif"],
  ["Courier New",     "courier new, courier, mono"],
  ["Georgia",         "Georgia, Times New Roman, Times, Serif"],
  ["Tahoma",          "Tahoma, Arial, Helvetica, sans-serif"],
  ["Times New Roman", "times new roman, times, serif"]
]

  • fontsizes : 字号列表

    二维数组
默认值 [
  ["特小","xx-small",	1],
  ["较小","x-small",	2],
  ["小",	"small",	3],
  ["中",	"medium",	4],
  ["大",	"large",	5],
  ["较大","x-large",	6],
  ["特大","xx-large",	7]
]

  • toolbar : 设置工具栏模式

    值可在字符串或数组,用于设置工具栏图标的组合. 默认值是 default。
当是字符串是可选值有 default|admin|mini|message|bbs|full|user。分别对应一些常用的工具栏组合模式。
点击查看示例
当是数组是，可以选择工具栏图标组合。点击查看示例

  • toolbarRight : 设置右边工具栏

    空值或数组。空值是，值根据 toolbar 的值而定。 点击查看示例
当是数组是，可以选择工具栏图标组合。点击查看示例

  • advToolbar : 大图标时左侧工具栏

    当是数组是，可以选择工具栏图标组合。点击查看示例

  • advToolbarRight : 大图标时右边工具栏

    当是数组是，可以选择工具栏图标组合。点击查看示例

  • advToolbarMode : 初始状态是否是大图标形式

    默认值 false.

  • showToolbarGroupHandle : 是否显示工具栏分组记号

    默认值 true.

  • autoPopImagePanel : 是否显示图片面板

    默认值 true.

  • autoPopTablePanel : 是否显示表格操作面板(插入行/列,删行/列/表格)

    默认值 true.

  • tplFlash : Flash代码模板

    插入Flash代码模板.支持宏变量  Flahs源址  宽（单位像素）  高（单位像素）

  • tplFlv : Flv播放器代码模板

    插入Flv视频代码模板.支持宏变量  Flv源址  宽（单位像素）  高（单位像素）  自动播放

  • tplRm : RM播放器代码模板

    插入Rm视频代码模板.支持宏变量  Rm源址  宽（单位像素）  高（单位像素）  自动播放

  • tplVideo : 一般视频播放器代码模板

    插入一般视频代码模板.支持宏变量  视频源址  宽（单位像素）  高（单位像素）  自动播放

  • tplMusic : 音频播放器代码模板

    插入音频代码模板.支持宏变量  音频源址  自动播放

  • tplSplitPage : 分页代码模板

    分页符的代码模板

  • uploadFieldName : 上传表单名

    上传的表单变量名。 默认值 file。
在服务器端php用 $_FILE['file'] 变量.

  • imageUploadUrl : 上传图片的接收网址或上传界面网址

    如这个值的第一个字符是"#"即表示自定义上传界面，否则当作接收上传文件的处理程序网址.
非定义接收网址时,返值为JSON字符串, 对不符合的数据将提示出错. 
JSON数据为:{url:'网址', error:'出错代码, 成功时为0,非0表示出错',message:'出错提示信息, 为空时出错不提示'}.
点击查看接收上传示例 
点击查看自定上传界面示例

  • imageFileTypes : 允许上传的图片文件后缀

    默认值 "*.jpg;*.gif;*.png;*.jpeg"

  • imageFileSize : 允许上传图片文件大小

    默认值 "2MB"

  • directInsertUploadImage : 设置上传后是否直接插入内容中

    默认值 false

  • linkUploadUrl : 上传链接文件的接收网址或上传界面网址

    请参看 imageUploadUrl 相关说明

  • linkFileTypes : 允许上传的链接文件后缀

    默认值 "*.*"

  • linkFileSize : 允许上传链接文件大小

    默认值 "2MB"

  • flashUploadUrl : 上传SWF的接收网址或上传界面网址

    请参看 imageUploadUrl 相关说明

  • flashFileTypes : 允许上传的SWF文件后缀

    默认值 "*.swf"

  • flashFileSize : 允许上传SWF文件大小

    默认值 "2MB"

  • musicUploadUrl : 上传音频的接收网址或上传界面网址

    请参看 imageUploadUrl 相关说明

  • musicFileTypes : 允许上传的音频文件后缀

    默认值 "*.mp3;*.wma"

  • musicFileSize : 允许上传音频文件大小

    默认值 "2MB".

  • videoUploadUrl : 上传视频的接收网址或上传界面网址

    请参看 imageUploadUrl 相关说明

  • videoFileTypes : 允许上传的视频文件后缀

    默认值 "*.flv;.mp4;*.mkv;*.wmv;*.asf;*.avi;*.rmvb;*.rm"

  • videoFileSize : 允许上传视频文件大小

    默认值 "2MB".

  • enter2br : 设置按回车键时是否使用换行符来替代

    默认值 false

  • width : 设置编辑器的宽度

    默认等于textarea的宽度.

  • height : 设置编辑器的高度（不含工具栏）

    默认等于textarea的高度.

  • autoClean : 设置粘贴Word代码时，是否自动清理无用代码

    默认值 true

  • faceCount : 设置表情数量, 在face目录中

    默认值 55

  • css : 内容设计辅助的相关样式网址

    必须用完整的绝对路径.

  • cssCode : 内容设计辅助的相关样式代码

    例:{cssCode:"body:font-size:12px;img:border:1px solid #666;"};

  • bgColor : 设置可视化编辑时, 编辑器背景色

    空值是为白色

  • textColor : 设置可视化编辑时, 文字色

    空值为黑色.

  • autoPopImagePanel : 设置是否自动弹出图片面板

    默认值 true.

  • autoPopTablePanel : 设置是否自动弹出表格相关面板(插入行/列,删除行/列/整表)

    默认值 true.

  • typesetImageTextCenter : 设置自动排版图片下方的文字居中

    默认值 true

  • typesetIndentParagraph : 自动排版段落首行缩进2字

    默认值 false

  • typesetImageBorder : 自动排版图片添加边框

    默认值 true

  • baseHref : 内容的相对网址

    用于解决前后台内容的相对网址不同.

  • resize : 设置允许修改编辑器尺寸

    默认值 true
可选值 [0|1|true|false|'h'] 
0或false时，禁止用户修改编辑器尺寸
1或true时，允许宽高
'h'时，只允许修改高度（v2.2.5版以上有效）

    相关说明正在编写中...

  • onkeydown : 设置编辑器按下按键时的回调函数

    相关说明正在编写中...

  • onkeyup : 设置编辑器释放按键时的回调函数

    相关说明正在编写中...

  • oninit : 设置编辑器初始化时的回调函数

    相关说明正在编写中...

  • ongetcontent : 设置取代码时的事件回调函数

    相关说明正在编写中...

  • onsetcontent : 设置代码时的事件回调函数

    相关说明正在编写中...

  • debug : 设置进入调试模式

    默认值 false
这个用于调试上传。
