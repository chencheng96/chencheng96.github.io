printf(格式控制,输出表列)函数输出数据

对不同类型的数据要指定不同的格式声明

  • d格式符（同i格式符）用来输出一个有符号的十进制整数。%5d指定输出数据占5列
  • c格式符用来输出一个字符。%5c指定输出数据格式为字符前面4个空格
  • s格式符用来输出一个字符串。%s
  • f格式符用来输出实数（单双精度，长双精度），以小数形式输出。%f。%7.2f制定输出数据占7列其中小数占2列。
  • e,E格式符以指数形式输出实数。%13.2e保留2为小数的指数形式，输出总长度为13，不足补空格。
  • o, x(X)格式符分别以八进制和16进制形式输出整数。
  • u格式符输出无符号型（unsigned）数据，以十进制输出。
  • g,G格式符输出浮点数，系统自动选用f格式或e格式，选择其中长度较短的格式输出。

输出%用%%

scanf(格式控制,地址表列)函数输入数据

格式字符与printf函数相同。输入字符串，在输入时以非空白字符开始，以第一个空白字符结束。

scanf函数中的格式控制后面应是变量地址，而不是变量名。在输入数据时要和scanf函数引号内的内容一一对应。scanf("a=%f",&a);应输入：a=1

格式附加字符l用于输入长整型数据及double型数据（%lf, %le）

格式附加字符h用于输入短整型数据（%hd, %ho）

函数 

void printstr()    //定义函数
{
    printf("this is function\n");
}

若函数位置在调用他的函数的后面，应在主调用函数中对被调用函数做声明：void printstr();

数组元素做实参时，是值传递。数组名作函数实参是，向形参（数组名或指针变量）传递的是数组首元素的地址。

二维数组的传递，形参数组定义时可以省略第一维的大小。

变量的存储方式

  • 自动变量（auto变量，默认类型）
  • 静态局部变量（static局部变量），函数调用后不消失，下一次进入函数变量值继承上次结果。
  • 寄存器变量（register变量）

全局变量

在两个文件中可以用 extern A;来将其他文件定义的变量A扩展到本文件。若在定义A是用static修饰，则不能被其他文件引用。

内部函数

两个文件中，可以用过extern（extern void printstr();）来声明外部函数以供本文件调用。若在定义内部函数是加上static修饰，则该函数不能被其他文件所调用。
