JavaScript,RegExp对象,正则表达式

正则表达式用来表示指定字符串在另一个字符串出现的情况。JavaScript对正则表达式的支持是通过RegExp对象来实现的。

RegExp的对象的构造函数可以带一个或两个参数，第一个参数是描述需要进行匹配的字符串，第二个参数指定了处理命令。

创建RegExp对象

使用new操作符创建RegExp对象

var r=new RegExp();  //创建一个RegExp对象

普通的字符串是最基本的正则表达式，比如匹配“tom”，可以定义正则表达式：

var r=new RegExp("tom"); //只匹配第一个"tom",且区分大小写

var r=new RegExp("tom","g");//“g”是global的缩写，表示要查找字符串中出现的全部“tom”。

var r=new RegExp("tom","ig");//查找字符串中出现的全部“tom”,且不区分大小写

RegExp对象的方法

  • test()方法：用于查找字符串中的指定值，如果找到返回true，否则返回false。
  • exec()方法：用于查找字符串中的指定值，如果找到返回被找到的字符串，否则返回null。
  • compile()方法：可以改变查找模式，也可以添加或删除第二个参数。

<script type="text/javascript">
var r=new RegExp("tom");
document.write(r.test("tomabc")+"</br>");//返回true
document.write(r.exec("tomabc")+"</br>");//返回tom
document.write(r.compile("h")+"</br>");//修改被查找的字符串为“h”
document.write(r+"</br>");  //返回null
document.write(r.exec("tomabc")+"</br>");  //返回null
</script>

正则表达式的元字符

正则表达式的一般形式：

/字符串/

位于/之间的部分就是要匹配的字符串。

元字符指在正则表达式中具有特殊意义的专用字符，可以用来规定前导字符（位于元字符前面的字符）在目标字符串中的出现模式。常见的有+，*，？

  • +：规定前导字符必须在目标字符串连续出现一次或多次
  • *：规定前导字符必须在目标字符串中出现零次或多次（任意次）
  • ?：规定前导字符必须在目标字符串中连续出现零次或一次

/fo+/：表示可以匹配在字母f后面连续出现一个或多个字母o的字符串

/fo*/：表示可以匹配在字母f后面连续出现零个或多个字母o的字符串

/fi?/：表示可以匹配字母f后面出现0次或一次字母i

如果要使用元字符本身，可以使用转义字符\，如\*得到*。

有时候知道要匹配多少个字符，可以在正则表达式中指定字符出现的次数。

  • {n}：n是一个正整数，匹配确定n次       a{2}表示要出现两个字母a。
  • {n,}：n是一个正整数，白色至少匹配n次         a{2,}表示a至少要出现2次。
  • {n,m}：m,n为正整数，n<=m，求至少匹配n次，最多匹配m次。      a{1,3}表示a可以出现1~3次。

几个其他的重要元字符的使用方法

.：用于匹配除换行符以外的所有字符

\s：匹配单个空格符，包括Tab键和换行符。

\S：匹配除单个空格符之外的所有字符。

\d：匹配0~9的数字

\w：匹配字母，数字或下划线

\W：匹配所有与\w不匹配的字符

可以把\s和\S已经\w和\W看做互为逆运算。

正则表达式的定位符

定位符用于规定匹配字符串在目标字符串中的出现位置，较为常见的有：^,$,\b,\B。

  • ^：规定匹配字符串必须出现在目标字符串的开头；
  • $：规定匹配字符串必须出现在目标字符串的结尾；
  • \b：规定匹配字符串必须出现在目标字符串的开头或结尾的两个边界之一；
  • \B：规定匹配字符串必须出现在目标字符串的开头或结尾的两个边界之内。

可以把^和$以及\b和\B看做互为逆运算。

/^hell/：匹配以hell开头的所有字符串

/ar$/：匹配以ar结尾的字符串

/\bfat/：匹配以fat开头或结尾的字符串

/man\B/：匹配中间含有man的字符串，但是开头和结尾不能是man

正则表达式的范围

正则表达式允许匹配字符串指定一个范围而不只局限于具体的某个字符。

  • /[a-z]/：表示a~z范围内的任何一个小写字母
  • /[A-Z]/：表示A~Z范围内的任何一个大写字母
  • /[0-9]/：表示0~9范围内的任何一个数字
  • /([a-z][A-Z][1-9])/：表示任何由字母和数字组成的字符串

()把三个字符串条件组合在一起，()符号包含的内容必须同时出现在目标字符串中，“aAa”不符号条件，因为最后一个符号不是数字。

  • |：在正则表达式中实现“或”运算的效果，使用|管道符   /a|b|c/表示匹配字母a或b或c
  • [^]：规定目标字符串中不能存在某个字符，可以使用否定符[^]       /[^a-c]/表示目标字符串中可以与除a,b,c之外的任何字符想匹配。

匹配日期的一个实例

<script type="text/javascript">
function isvalid(s){
var d=/(?:0[1-9]|[12][0-9]|3[01])\/(?:0[1-9]|1[0-2])\/(?:19|20\d{2})/;
return d.test(s);
}
window.alert(isvalid("05/05/2010"));//true
window.alert(isvalid("5/5/2010"));//false
</script>
