jsp使用kindeditor的上传路径,图片空间路径配置以及引用

jsp,kindeditor,路径

1. 需要将kindeditor-4.1.10/jsp/lib下的3个jar包导入到apache服务器的lib中

2. 导入的两个js文件目录路径为相对于当前目录的路径！

<script charset="utf-8" src="../../kindeditor-4.1.10/kindeditor.js"></script>
<script charset="utf-8" src="../../kindeditor-4.1.10/lang/zh_CN.js"></script>
<script>  
        KindEditor.ready(function(K) {  
              K.create('#editor_id', {  
                uploadJson : '../../kindeditor-4.1.10/jsp/upload_json.jsp',  
                fileManagerJson : '../../kindeditor-4.1.10/jsp/file_manager_json.jsp',  
                allowFileManager : true  
              });  
        });  
</script> 
<textarea id="editor_id" name="content" style="width:96%;height:350px;visibility:hidden;">${WenZhang.content}</textarea>

3. kindeditor-4.1.10/jsp/upload_json.jsp配置路径

//文件保存目录路径
String savePath = pageContext.getServletContext().getRealPath("/") + "File/ForKindEditor/";
//文件保存目录URL
String saveUrl ?= request.getContextPath() + "/File/ForKindEditor/";

4. kindeditor-4.1.10/jsp/file_manager_json.jsp配置路径

//根目录路径，可以指定绝对路径，比如 /var/www/attached/
String rootPath = pageContext.getServletContext().getRealPath("/") + "File/ForKindEditor/";
//根目录URL，可以指定绝对路径，比如 http://www.yoursite.com/attached/
String rootUrl ?= request.getContextPath() + "/File/ForKindEditor/";

/File/Image/ForKindEditor/为服务器站点根目录开始到图片目录的路径
