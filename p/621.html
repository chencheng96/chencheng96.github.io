JavaScript,窗口操作

window对象有可以对窗口自身进行控制的方法，包括新开窗口，关闭窗口，控制窗口位置和大小等。

打开窗口open()

window.open(url,windowname,features,replace)

url：打开新窗口中显示文档的网址，如果参数被省略，为空或者null时，打开一个空白窗口。

windowname：表示新开窗口的名字。如果指定的是一个已经存在的标签，则打开那个已经存在的标签而不是一个新标签。

features：特征列表，标明窗口的大小，是否显示工具栏，菜单栏，状态栏，默认值为no

replace：只有在windowname参数存在时才起作用，是一个布尔值。如果参数是true，用第一个参数值替换该窗口浏览历史的当前页；如果是false，在该窗口浏览历史中创建一个新项。

open()方法返回值是代表新创建的窗口的window对象。

新窗口的特征参数：

width：窗口宽度

height：窗口高度

location：是否显示地址栏，参数值为yes或no

menubar：是否显示菜单栏，参数值为yes或no

resizable：是否可以调整窗口大小，参数值为yes或no

scrollbars：是否显示滚动条，参数值为yes或no

status：是否显示状态栏，参数值为yes或no

toolbar：是否显示工具栏，参数值为yes或no

关闭窗口close()

关闭一个窗口。一个窗口关闭了，它的window对象仍然存在，可以检测它的closed属性，属性值为true表示关闭了。

<script type="text/javascript">
var win;
function openwin(){win=window.open("","","width=300,height=300");}
function closewin(){
if(win.closed)window.alert("已关闭窗口");
else win.close();
}
</script>

滚动scrollTo()&scrollBy()

  • window.scroll(x,y)：将窗口中显示的文档滚动到指定的绝对位置，位置由（x,y）决定，文档左上角坐标为(0,0)。
  • window.scrollTo(x,y)：和window.scroll(x,y)完全相同
  • window.scrollBy(x,y)：将窗口中显示的文档滚动到指定位置，x值为正，文档向右滚动。如果参数y为正，文档向下滚动。

调整窗口大小resizeTo()&resizeBy()

  • window.resizeTo(x,y)：将窗口调整到一个绝对大小，x表示窗口宽度，y表示窗口高度
  • window.resizeBy(x,y）：保存当前窗口大小不变的情况下，增加或减少相应的宽度和高度。
