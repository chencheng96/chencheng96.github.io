JavaScript,语句

选择语句

if语句

if(逻辑表达式){语句块}else{语句块}，if语句可以嵌套。

<script type="text/javascript">
var d=new Date();   //创建一个日期对象
var hour=d.getHours(); //调用getHours()方法，获得当前计算机时间，保持在变量hour中
if(hour>5&&hour<12){window.alert("早上好！");}
else if(hour>12&&hour<=18){window.alert("下午好！");}
else{window.alert("晚上好！");}
</script>

switch语句

多分支选择语句，先计算变量的值，然后再与常量表达式比较，如果相同则执行该语句。如果都不相同，则执行default后的语句块。

switch(变量)

{

case 常量表达式1: 语句块1; break;

case 常量表达式2:语句块2; break;

...

default: 语句块;

}


循环语句

while语句

先判断，后执行

while(逻辑表达式){语句块}

do...while语句

先执行循环体再判断循环条件，尽量不要使用

do{语句块}while(逻辑表达式)

for语句

for(循环变量初始化;循环条件;循环变量增值;){语句块}

for...in语句

可以遍历对象中的所有属性或数组中的所有元素，也常用来为对象的所有属性赋值。

for(变量 in 对象){语句块}

<script type="text/javascript">
var arr=["java",100,true];
var i;
for(i in arr){window.alert("数组中元素的值分别是"+arr[i]);}
</script>

跳转语句

break语句

跳出循环体或结束switch语句。在嵌套的循环语句中，break语句只能跳出离该语句最近一层的循环语句，而不是所有循环语句。

break语句也可以通过借助标签语句跳转到一个指定的循环之外

<script type="text/javascript">
out: //标签语句
for(var i=0;i<4;i++){

document.write("循环开始</br>");

document.write("变量i的值是："+i+"</br>");

for(var j=0;j<10;j++){

document.write(" 变量j的值是："+j+"</br>");

if(j==1){break out;};

}

}
document.write("最终变量i的值是："+i+",j的值是："+j);
</script>

continue语句

作用是结束本次循环,跳过循环体中下面尚未执行的语句，接着进行下一次是否循环的判断。

continue语句只结束本次循环，而不是终止整个循环的执行；而break语句则是结束整个循环过程，不再判断执行循环的条件是否成立。

<script type="text/javascript">
for(var j=0;j<=5;j++){
if(j==3){continue;};  //终止本次循环，继续下一次循环的判断
document.write(j);
}
</script>

异常处理语句

throw语句

抛出一个异常，通知发生了异常情况或错误。只是抛出一个异常，但对这个异常并不处理。

throw 表达式;

<script type="text/javascript">
var a=10;
if(a>10){throw new Error("错误，a的值不能大于10");}
</script>

try...catch...finally语句

try{语句块}catch(e){语句块}finally{语句块}

用来处理抛出的异常

<script type="text/javascript">
var x=window.prompt("请输入1~5之间的整数","") //弹出一个输入窗口，等待输入，并把这个值赋给变量x
try{if(x>5){throw "Err1"}else if(x<1){throw "Err2"}
}
catch(er){if(er=="Err1"){window.alert("输入的整数大于5");}
if(er=="Err2"){window.alert("输入的整数小于1");}
}
finally{}
</script>
