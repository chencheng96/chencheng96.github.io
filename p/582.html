盒模型

页面上的每一个元素都被浏览器看成是一个矩形的盒子，网页就是有许多盒子通过排列堆积而成（上下排列，并列排列，嵌套排列）

一个盒子模型由4部分组成

  • content
  • border（边框）指盒子周边的直线，四条边都可以设置边框线
  • padding（填充）内容与边框线之间的空间
  • margin（边距）指一个标签与另一个标签之间的间隔

在css中分为block box（块级盒子）和inline box（内联盒子），它们分别对应于块级标签和内联标签。


块级标签（div）

<div></div>之间的内容为一个独立的对象，内部可以容纳段落标题，表格，图片等HTML元素。可以给每一个<div>标签添加一个类选择器或者ID选择器，用于css控制。每个网页只能使用一个ID选择器，如果一个元素多次出现要选择类选择器。利用浮动或绝对定位将这些div标签固定在网页上。

<div class="border">这是容器的内容</div>

<div id="border">这是容器的内容</div>


<span>标签用法和<div>完全相同。不过span是一个内联标签，在它的前后不会换行。

<span>可以包含在<div>中做它的子元素，反过来则不成立。


边框（border）

可以通过三个属性进行控制：border-style（边框样式属性），border-width（边框宽度属性），border-color（边框颜色属性）

border-style的属性

border-left-style 设置左边框，一般单独设置左边框样式使用

border-right-style 设置右边框，一般单独设置右边框样式使用

border-top-style 设置上边框，一般单独设置上边框样式使用

border-bottom-style 设置下边框，一般单独设置下边框样式使用,有时可将下边框样式作为css下划线效果应用。

也可以一起进行设置

border-style: solid;               取一个值，四条边同一属性

border-style: solid dotted;             取两个值：上下 左右的顺序

border-style: solid dotted double;         取三个值：上 左右 下的顺序

border-style: solid dotted double dashed;          取四个值：上 右 下 左的顺时针顺序

border-style属性取值：

值 描述
none 定义无边框。
hidden 与 "none" 相同。不过应用于表时除外，对于表，hidden 用于解决边框冲突。
dotted 定义点状边框。在大多数浏览器中呈现为实线。
dashed 定义虚线。在大多数浏览器中呈现为实线。
solid 定义实线。
double 定义双线。双线的宽度等于 border-width 的值。
groove 定义 3D 凹槽边框。其效果取决于 border-color 的值。
ridge 定义 3D 垄状边框。其效果取决于 border-color 的值。
inset 定义 3D inset 边框。其效果取决于 border-color 的值。
outset 定义 3D outset 边框。其效果取决于 border-color 的值。
inherit 规定应该从父元素继承边框样式。


border-width的属性

border-left-width 设置左边框，一般单独设置左边框样式使用

border-right-width 设置右边框，一般单独设置右边框样式使用

border-top-width 设置上边框，一般单独设置上边框样式使用

border-bottom-width 设置下边框，一般单独设置下边框样式使用,有时可将下边框样式作为css下划线效果应用。

也可以同style属性相同的顺序一起进行设置

border-width的属性取值

值 描述
thin 定义细的边框。
medium 默认。定义中等的边框。
thick 定义粗的边框。
length 允许您自定义边框的宽度。
inherit 规定应该从父元素继承边框宽度。

border-color的属性

border-left-color 设置左边框，一般单独设置左边框样式使用

border-right-color 设置右边框，一般单独设置右边框样式使用

border-top-color 设置上边框，一般单独设置上边框样式使用

border-bottom-color 设置下边框，一般单独设置下边框样式使用,有时可将下边框样式作为css下划线效果应用。

也可以同style属性相同的顺序一起进行设置

border-color的属性取值

值 描述
color_name 规定颜色值为颜色名称的边框颜色（比如 red）。
hex_number 规定颜色值为十六进制值的边框颜色（比如 #ff0000）。
rgb_number 规定颜色值为 rgb 代码的边框颜色（比如 rgb(255,0,0)）。
transparent 默认值。边框颜色为透明。
inherit 规定应该从父元素继承边框颜色。

上述3种属性border-style（边框样式属性），border-width（边框宽度属性），border-color（边框颜色属性）统一设置

border: 2px solid #f00;

这三种属性的顺序没有先后

也也可以每个边单独设置

border: 2px solid #f00;

border-top: 4px solid #f66;

根据就近原则后一行的代码会覆盖border关于top的设置。



边距（margin）

在元素与元素之间增加空间包括

margin-top

margin-bottom

margin-left

margin-right

margin复合边距属性  （同style属性相同的顺序一起进行设置）

边距的值有三种：长度（像素,em），百分比，auto（默认值）


填充（padding）

在内容和边框之间增加空间，属性和用法和margin完全相同。

如果css属性值为0，就不要添加度量单位了。

<style type="text/css">

.m1{

background: #ccc;

width: 200px;

height: 100px;

margin: 0 auto;

}

.m2{

border: 2px solid #333;

background: #fff;

width: 100px;

height: 50px;

margin: 10px;

float: left;

padding: 10px;

}

</style>

</head>

<body>

<div class="m1">

<div class="m2">边距和填充都是10像素</div></div>

</body>


边距折叠

当元素的底部边距（bottom margin）碰到另一个元素的顶部边距（top margin）时，浏览器取较大的那个值进行处理。

通过给元素添加边框或填充使两个margin分割开。

水平的边距不会发生这种折叠


width（宽度）和height（高度）

width和height指的是内容区域的宽度和高度，使用css的长度单位：像素（px），百分比，em（字号的倍数），auto（和父元素宽度相同）

可以设置width和height的最大值和最小值，max-width,max-height,min-width,min-height.


盒子模型的实际width（宽度）和height（高度）

width和height属性设置了内容区域，实际宽度和高度是margin,border,padding,width/height的总和。
