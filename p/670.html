上篇文章中有说到document.execCommand("Copy"); 执行浏览器复制命令的一种使用实例，这里复制的是文本区域。同样通过document.getElementById("biao1")获取表单元素的ID也可以获得text即文本内容到剪切板。

<script type="text/javascript">
function copyUrl2()
{
var Url2=document.getElementById("biao1");
Url2.select(); // 选择对象
document.execCommand("Copy"); // 执行浏览器复制命令，复制上一语句选定的对象
alert("已复制好，可贴粘。");
}
</script>
<textarea cols="20" rows="10" id="biao1">用户定义的代码区域</textarea>
<input type="button" onClick="copyUrl2()" value="点击复制代码" />


但是上述方法对于表单域hidden是无效的，可能是因为无法使隐藏域处于选定状态。但是通过下面方法可以复制隐藏域到剪切板，但是局限性很大（只能在IE浏览器中使用）

<script type="text/javascript">
function copyUrl2()
{
var Url2=document.getElementById("biao");
var clipBoardContent=Url2.value;// 选择对象
window.clipboardData.setData("Text",clipBoardContent);        //复制clipBoardContent的内容到剪切板。
alert("已复制好，可贴粘。");
}
</script>
<input type="hidden"  id="biao" value="点击复制代码" />
<input type="button" onClick="copyUrl2()" value="点击复制代码" />


对于在其他浏览器中复制隐藏域中的内容到剪切板的简单方法还没有找到。但是对于Javascript中document.execCommand()的用法还需要进一步的了解。
