命令行下运行 MongoDB 服务器

为了从命令提示符下运行 MongoDB 服务器，你必须从 MongoDB 目录的 bin 目录中执行 mongod.exe 文件。 

D:\Program Files\MongoDB\Server\3.2\bin>mongod --dbpath F:\mongodb\db

其中F:\mongodb\db是存放数据库数据的位置

连接MongoDB服务器

在新的cmd命令框中建立连接（不能关闭上述打开服务器的窗口），之后可以对数据库进行有关操作 

D:\Program Files\MongoDB\Server\3.2\bin>mongo.exe

将MongoDB添加到Window服务

创建一个配置文件mongod.cfg位于指定位置（建议放在MongoDB的安装目录下），其中指定 systemLog.path 和 storage.dbPath。具体配置内容如下： 

systemLog:
    destination: file
    path: c:\data\log\mongod.log
storage:
    dbPath: c:\data\db
通过执行mongod.exe（需要使用管理员权限！），使用--install选项来安装服务，使用--config选项来指定之前创建的配置文件。 
D:\...安装目录\3.2\bin>mongod --config "D:\Program Files\MongoDB\Server\3.2\mongod.cfg" --install
执行结果可以在mongod.cfg中指定的log文件中看到。

安装之后启动或关闭以及移除服务方法

启动MongoDB服务：net start MongoDB
关闭MongoDB服务：net stop MongoDB
移除 MongoDB 服务：D:\Program Files\MongoDB\Server\3.2\bin\mongod.exe --remove
