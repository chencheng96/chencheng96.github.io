打开窗口：
start cmd /k echo Hello, World!  #  # 执行完毕以后，新开的窗口不会自动关闭

start cmd /C pause    # 执行完毕以后，新开的窗口会自动关闭


可以通过 "taskkill /f /im cmd.exe" 来关闭所有的cmd窗口

其实此命令可以 用来关闭任何任务及窗口 只要在cmd 中输入 “ tasklist ”   会显示出所有的正在运行的任务，
找到对应要关闭的 任务  通过这个命令 “ taskkill  /f /im 映像名称” （例如 taskkill  /f /im  cmd.exe）
