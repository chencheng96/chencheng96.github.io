ASP,组件

ASP提供的组件可进一步增强脚本的功能，用户可以自己编写组件。

Ad Rotator（图片交替显示组件）

   使用该方法在每次刷新页面会更换图片和对应了URL链接，需要创建一个以.txt后缀的轮显列表。

Content Rotator（字符串轮显组件）

   该方法轮显的内容是字符串，同Ad Rotator一样，也需要配置.txt的轮显列表。

Browser Capabilities（浏览器功能组件）

   检测正在浏览的某网页的浏览器是否支持某些功能。

   Set 对象=Server.CreateObject("MSWC.BrowserType")

   可以将浏览器及其功能写到一个列表中，该文件名为"browscap.ini"。

File Access（文件存取组件）

   对文件系统进行操作，可以取得和处理服务器上所有驱动器的相关信息；取得服务器上所有文件夹和子文件夹的相关信息；可以处理文件夹内的文件。

Content Linking（文件导航组件）

   用来建立快速而简单的导航系统。创建Content Linking组件后将返回一个用来对页面列表进行导航的NextLink组件。

   Set cl对象=Server.CreateObject("MSWC.NextLink")
