JavaScript,链接对象,锚对象

JavaScript会自动创建一个links[]数组，数组中的每个元素对应网页中的<a>标签。<area>标签也可以在document对象的links[]数组中创建链接对象。

链接对象（link）表示URL，还有location对象的所有属性。引用方式：document.links[i]

链接对象的属性

链接对象引用的是网页文档的超级链接，包括<a>和</a>标签之间的所有内容。

属 性 说 明
hash 链接URL中锚的部分并包括“#”符号
host 链接URL中的主机名称和端口号
hostname 链接URL中的主机名称
href 完整的链接URL
pathname 链接URL中的路径名部分
port 链接URL中的端口号
protocol 链接URL中的协议部分并包括“：”
search 链接URL中的条件部分并包括“”符号
target 链接的目标窗口打开方式
text 是Netscape浏览器支持的属性，主要用来显示链接对象中的超级链接文字，这个属性是只读的，不能修改
innerText IE浏览器支持的属性，等同于text


<a href="#1">the1</a>
<a href="#2">the2</a>
<a href="#3">the3</a>
<a href="#4">the4</a></br>
<script type="text/javascript">
for(var i=0;i<document.links.length;i++){document.write(document.links[i].href+"</br>");} //显示网页中所有链接
</script>

链接对象的事件

 事件 说明  事件  说明 
click 在链接单击鼠标左键时激发的事件 mousedown 在链接上按下鼠标键时激发的事件
dbclick 在链接上双击鼠标时激发的事件 mouseup 在链接上释放鼠标键时激发的事件
keydown 按下键盘上的键时激发的事件 mouseover 把鼠标移动到链接上时激发的事件
keypress 按下并释放键盘上的键时激发的事件 mousemove 在链接上移动鼠标时激发的事件
keyup 释放键盘上的键时激发的事件 mouseout 把鼠标从链接上移开时激发的事件

锚对象

只有在<a>标签中才能创建一个锚，所以锚对象定义的唯一属性是name。加载文档时会自动创建一个anchors[]数组。anchor[]数组中的元素就是网页上的锚。调用方式：document.anchors[i]。

锚对象的属性

name：返回锚的名称，就是<a>标签的name属性值

text：Netscape浏览器支持的属性，主要用来显示链接对象中的超级链接文字，这个属性是只读的

innerText：IE浏览器支持的，等同于text

锚对象和链接对象都是引用<a>标签。<a>标签中必须有href属性才能创建链接对象，必须有name属性才能创建锚对象。如果两种属性都有，可以同时创建两个对象。
