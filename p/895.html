弹出窗口（浮层）

弹窗本质是html页面的某层。某事件发生后执行相应的js程序，使某容器的display属性为block。对于显示的内容可以是在页面已经加载的，也可以是通过js动态生成的。对于弹窗显示的位置，如果不是针对某元素可以设置浮在页面的某个地方（既然已经绝对地址了，一般就放在页面代码的最下面）。

<div id="floatDiv" style="display:none; top: 155px; left: 300px; width: 380px;position: fixed; display: block; background: #F0F3F5; padding: 10px 10px 10px 10px; border: 1px solid #d6d6d6; ">

弹窗的关闭

一般在右上角用一个标记，然后添加一个onclick事件，当事件发生时，设置上面弹窗的的display属性为none。

弹窗表单的提交

对于弹窗来说，提交数据一般是采用异步提交的方法。当然可以用常用的两种方法，ajax和隐藏区域（伪异步）提交相关数据。

信息提交后的反馈

对于提交信息的返回值进行判断，然后显示相应的提示信息。可以在原浮动窗口上实现此种变化。对于表单的提交，可以显示是否提交成功的反馈。在显示信息时，可以对原表单信息进行隐藏，而显示相关的提示信息。

点击按钮切换页面内容

这里是在页面加载全部所需的数据，当点击切换时，调用相应的js函数，对显示内容的display属性进行设置。对于异步加载内容并显示，在另一个模块（ajax）中已经进行了说明。

js代码定时更新

js中按照指定的周期循环调用函数。self.setInterval("clock();",1000);//每1秒调用一次clock()函数

JS的常用输出消息的四个方法

1)JavaScript 语句告诉浏览器向网页输出消息
document.write("你好,JS!!!");
2) alert() 弹出消息框
3) true | false  confirm() 弹出消息框,返回true或false询问框
<script type="text/javascript">
//使用var关键字声明变量   
 var result = confirm("你确定吗?");
 document.write("你选择的是:"+result); 
</script>
4) prompt() 用于提示用户输入消息
<script type="text/javascript">
  var name = prompt("请输入你的名字:","");
  document.write("你输入的是: "+name);
</script>
