JavaScript,文本框

文本框分为单行文本框（包括密码框）和多行文本框两种。

文本框对象属性

文本框对象（text对象），多行文本框对象（textarea对象），密码对象（password对象）。

accesskey：返回或设置文本框的快捷键

defaultvalue：返回或设置文本框的初始文本。对于单行文本，由input标签的value属性值决定；对于多行文本，该属性值由<text>和</text>标签之间的文本决定

disabled：返回或设置文本框是否被禁用，属性值为布尔值

form：返回包含文本框元素的form对象的引用

id：返回或设置文本框的id属性

maxlength：返回或设置文本框可输入文字的最大字符数

name：返回或设置文本框名称

readonly：返回或设置文本框是否为只读，属性值为布尔值

size：返回或设置单行文本框和密码框的宽度大小

tabindex：返回或设置文本框的Tab键顺序

type：返回或设置文本框类型，单行文本框返回text，多行返回textarea，密码框返回password

value：返回或设置文本框的值

rows：返回或设置多行文本框的高度

cols：返回或设置多行文本框的宽度

文本框对象方法

blur()：将焦点从文本框移开

click()：模拟文本框被鼠标点击

focus()：将焦点赋给文本框

select()：选中文本框中的文字

文本框对象事件

三种文本框对象响应的事件是相同的。

blur：当焦点从文本框移开时激发的事件

change：当文本框中的内容改变时激发的事件

click：单击文本框时激发的事件

dbclick：双击文本框时激发的事件

focus：当焦点赋给文本框时激发的事件

keydown：用户按下键盘键时激发的事件

keypress：按下并释放键盘键时所激发的事件

keyup：释放键盘时所激发的事件

mousedown：按下鼠标键（并没有释放）时所激发的事件

mousemove：移动鼠标时激发的事件

mouseover：鼠标从文本框对象上移开时所激发的事件

mouseup：释放鼠标键时所激发的事件

mouseout：鼠标从文本框对象上移开时激发的事件

select：选中文本时所激发的事件

selectstart：当文本框中的文字开始被选中时激发的事件

自动选择文本框中的文字

使用文本框的select()方法和mouseover事件相结合，可以在鼠标经过的文本框时自动选中文本框的内容，然后将文本框中的文字内容清除，方便用户输入。

<script type="text/javascript">
function cleartext(tb){
tb.focus();
if(tb.value==tb.defaultValue)tb.value="";
}
function test(tb){
if(tb.value.length<=6)tb.value="输入错误，请输入大于6个字符";
}
</script>
<form action="" method="post" name="forme1">
姓名：<input type="text" name="name" onmouseover="cleartext(this)" value="请在这输入"/>
<input type="submit" name="submit"  value="提交"/>
<input type="reset" name="reset" value="重置"/>
</form>
