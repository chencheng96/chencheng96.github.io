C++源程序文件扩展名是.cpp。可以使用Microsoft Visual Studio的集成开发环境（Integrated Develop Environment, IDE）开发。注释同C

#include//编译预处理命令，iostream是Visual C++.NET系统提供的头文件。声明输入输出的有关信息。

using namespace std;//命名空间 

cout << "" << a << endl;     //cout通过<<输出字符串""，变量a，和控制字符。endl用来换行。
cin >> a;    //cin通过>>输入内容给a。

基本数据类型有：int, short int, long int, unsigned. 浮点型float, double. 字符型char. 布尔型boolean。

构造数据类型有：数组，指针，结构体，共用体，枚举，类，空类型。

常量用const关键字进行定义。const double PI = 3.1415926;

#define命令定义一个常量（预处理）。#define PI 3.1415926

运算符

算数运算：+，-，*，/，%（取模）

++，--

&&，||，！

>,<,==,>=,<=,!=

位运算：x<>(右移，左按符号位补1或0), >>>(无符合右移左补0), &(与), |(或), ^(异或), ~(位反)

内置函数

求整数绝对值int abs(int x);    头文件cstdlib

占用内存字节数 sizeof(x);    头文件iostream


程序控制语句同C

函数的调用

函数的传值调用：主调函数把实参的值传送给被调函数的形参

函数的引用调用：int a; int &ra=a; //ra为引用变量，a为被引用变量，不同于取址运算符。ra和a的地址相同，共用同一个存储单元。用引用变量做函数的形参，则对形参的操作，实参也会发生相应的改变。对应函数void change(int &a_temp){a_temp=5;}则调用函数change(a);后a==5;

函数的嵌套调用和函数的递归调用

函数模板

由模板函数实现，调用时自动选择合适的参数类型。 

Template <class T>
void swap(T a, T b)
{
    T temp;
    temp=a;a=b;b=temp;
}

动态局部变量auto;静态局部变量static;

外部变量 extern int s; //定义外部变量s,改变s的值，其他程序中同名的全局变量会受到影响。

随机函数

rand()结合两个用于产生随机数种子的函数srand()（需要用到cstalib头文件）和time()（需要用到ctime头文件）才能产生不同序列的随机数。srand(time(0)); int n=rand();

函数的重载

重载函数需要在参数类型，参数个数或者参数顺序上有所不同。


数组，字符数组（字符串处理函数）

结构体struct，共用体union

枚举类型enum Color{red, blue, green};，其值为字符串对于位置的整数值

文件包含预处理

#include <文件名>

#include "文件名"

宏定义

无参数宏定义 #define PI  3.1415926

带参数宏定义 #define S(a,b)  a*b

宏定义的解除 #undef PI

指针，指针变量，字符串，数字，指针型函数，指向结构体的指针。
