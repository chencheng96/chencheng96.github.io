JavaScript,图像对象

图像（image）对象是document对象的子对象。

图像对象介绍

JavaScript加载网页文档时会自动创建一个images[]数组，这个数组中的每一个元素都代表网页中的一张图片。引用图片的方法：

document.images[i]   //如果网页有三张图片，那么images[1]表示第二个图片

document.imgname  //通过img标签的name属性值。

图像对象属性

除了基础document对象属性外，图像对象也有自己的属性，主要描述宽度，高度，边框等信息。

align：设置图片与内联内容的对齐方式

alt：设置或返回图片无法显示时的文字

complete：浏览器判断图片是否加载完整    这个是否涉及到懒加载？？？有待研究

border：设置或者返回图片周围的边框

height：设置或返回图片的高度

width：设置或返回图片的宽度

isMap：返回图像是否是服务器端的图像映射。  这个是否可以用做判断从数据库中拿到的数据是否正确？

useMap：设置或返回客户端图像映射的 usemap 属性的值。

id：设置或者返回图片的id  利用id操作图片

name：设置或者返回图片的name 

hspace：设置或者返回图片左或者右的空白部分

vspace：设置或者返回图片上或者下的空白部分

longDesc：设置或返回指向包含图像描述的文档的 URL。（即alt或者是有文字的图片中的文字）

lowsrc：设置或返回指向图像的低分辨率版本的 URL。

src：设置或返回图像的 URL。

<img src="https://953.jpg" hspace="30" vspace="10" border="5" name="tx" lowsrc="https://53.jpg" align="left">
<script type="text/javascript">
document.write("图像的宽度为"+document.tx.width+"像素</br>");
document.write("图像与文字水平方向间距是"+document.tx.hspace+"像素</br>");
document.write("图像的地址为"+document.tx.src+"</br>");
document.write("图像的低质量代替图像是"+document.tx.lowsrc);
</script>

图像对象事件

abort：用户放弃加载图像时激发的事件

click：在图像上单击鼠标左键时激发的事件

dbclick：在图像上双击鼠标时激发的事件

error：加载图像发生错误时激发的事件

load：加载图像成功时激发的事件

keydown：按下键盘上的键时激发的事件

keypress：按下并释放键盘上的键时激发的事件

keyup：释放键盘上的键时激发的事件

mousedown：在图像上按下鼠标键时激发的事件

mouseup：在图像上释放鼠标键时激发的事件

mouseover：把鼠标移动到图像上时激发的事件

mousemove：在图像上移动鼠标时激发的事件

mouseout：把鼠标从图像上移开时激发的事件

随机图像

有JavaScript产生一个随机数，再根据这个随机数显示图片，然后在网页上循环显示图片，显示顺序是随机的。

<script type="text/javascript">
function simg(){
var r=Math.random(); //产生0~1之间的随机数
var ran=Math.round(r*10); //转换成1~10之间的随机数
if(ran>5)ran=ran-5;
var arr=new Array();   //图像地址保存在数组中
arr[0]="http://www.iyaka.cn/advertiser/kindeditor/plugins/emoticons/images/0.gif";
arr[1]="http://www.iyaka.cn/advertiser/kindeditor/plugins/emoticons/images/1.gif";
arr[2]="http://www.iyaka.cn/advertiser/kindeditor/plugins/emoticons/images/2.gif";
arr[3]="http://www.iyaka.cn/advertiser/kindeditor/plugins/emoticons/images/3.gif";
arr[4]="http://www.iyaka.cn/advertiser/kindeditor/plugins/emoticons/images/4.gif";
arr[5]="http://www.iyaka.cn/advertiser/kindeditor/plugins/emoticons/images/5.gif";
document.tx.src=arr[ran];   //数组元素中的图像地址赋给图像的src属性
}
window.setInterval("simg()",2000); //设置两秒换一张图像
</script>

改变图像大小

在程序代码中修改图像宽度和高度的值来动态改变图像大小。

<script type="text/javascript">
function larger(){
document.images.tx.width=document.images.tx.width*1.5;
document.images.tx.width=document.images.tx.height*1.5;
}
function smaller(){
document.images.tx.width=document.images.tx.width*0.5;
document.images.tx.width=document.images.tx.height*0.5;
}
</script>
<img src="http://www.iyaka.cn/advertiser/kindeditor/plugins/emoticons/images/6.gif" name="tx"/>
<input type="button" value="放大图像" onclick="larger()"/>
<input type="button" value="缩小图像" onclick="smaller()"/>

显示默认图像

网页中加载图像出现错误（路径不正确或不存在）时，会激发error事件。可以用一张默认的图像代替。

<script type="text/javascript">
function show(image){image.src="http://www.iyaka.cn/advertiser/kindeditor/plugins/emoticons/images/1.gif";}
</script>
<img src="images/6.gif" onerror="show(this)"/>
