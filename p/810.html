可以从Sun的Java网站（http://www.oracle.com/technetwork/java/index.html）下载J2SE（Java开发包），然后安装，假设安装路径为D:\retainProfessional。然后在系统环境变量PATH最前面添加Java路径D:\retainProfessional\Jave\jdk1.6.0_10\bin。然后增加一个环境变量“JAVA_HOME”指向Java安装路径D:\retainProfessional\Jave\jdk1.6.0_10\。

1、首先安装JDK，然后配置java环境变量

（1）、变量名JAVA_HOME   变量值为  D:\Program Files (x86)\Java\jdk1.6.0_10

（2）、 变量名PATH 变量值为D:\Program Files (x86)\Java\jdk1.6.0_10\bin 或者 %JAVA_HOME%\bin

（3）、  变量名CLASSPATH   变量值为   .;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar


运行java和javac命令，出现以下提示说明jdk安装完成。


2、下载tomcat6 免安装版本，这里下载apache-tomcat-6.0.20

然后配置tomcat6环境变量：

（1）、 变量名CATALINA_HOME   变量值为  D:\apache-tomcat-6.0.20

（2）、 变量名TOMCAT_HOME     变量值为  D:\apache-tomcat-6.0.20

3、注册服务

解压版的  没有在Windows里 注册服务   必须要手动注册服务   不然无法运行tomcat6.exe

只能通过 startup.bat 启动Tomcat     修改文件D:\apache-tomcat-6.0.20\bin\service.bat

在rem Set default Service name（在此下添加下面一行）
set CATALINA_HOME=Tomcat安装目录或 % CD %


set SERVICE_NAME=Tomca6
set PR_DISPLAYNAME=Apache Tomcat 6

4、安装服务

运行cmd打开控制台，进入Tomat目录/bin文件夹，输入如下命令运行。
service.bat install
程序提示：The service 'Tomcat6（或者你修改一后的SERVICE_NAME）' has been installed
说明服务Tomcat已经被安装成功。下面是安装失败的提示

注意用管理员的身份运行cmd，否则会出现：Failed installing tomcat6 service

5、运行bin下的startup.bat启动服务

Tomcat6.0的目录结构

  • /bin：存放windows或Linux平台上启动和关闭Tomcat的脚本文件
  • /conf：存放Tomcat服务器的各种全局配置文件，其中包括server.xml（Tomcat的主要配置文件）、tomcat-users.xml和web.xml等配置文件
  • /lib：存放所需的所有jar文件（整合了原来tomcat5.5中的common/share/server三个目录下的所有jar）
  • /logs：存放Tomcat执行时的日志文件
  • /temp：存放Tomcat运行时所产生的临时文件
  • /webapps：Tomcat的主要Web发布目录，默认情况下把Web应用文件放于此目录（原/server/webapps下的manager等应用也已转移至此）
  • /work：Tomcat将JSP生成的Servlet源文件和字节码文件放到这个目录下
