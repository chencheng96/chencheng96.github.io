JavaScript,window时间间隔

用来设置代码的执行时间和执行方式

延迟代码执行setTimeout(codes,delay)

codes：表示代码的字符串，或函数名

delay：表示延迟的时间，单位为毫秒。

每次调用setTimeout()方法都会产生一个唯一的ID，可以通过clearTimeout()方法暂停setTimeout()方法还未执行的代码。

周期性代码执行setInterval(codes,interval)

codes：要周期执行的代码

interval：时间间隔，单位为毫秒

停止延迟性代码执行clearTimeout(id)

id：setTimeout()方法的返回值。

停止周期性代码执行clearInterval(id)

id：setInterval()方法的返回值。

<script type="text/javascript">
var id;
function timeout(){
id=setTimeout("window.alert('延迟2秒弹出的窗口！')",2000);}
function interval(){
setInterval(document.write(new Date()+" 间隔1秒执行一次代码"),1000);
}
function stop(){clearTimeout(id);}
</script>
<input type="button" value="延迟执行代码" onclick="timeout()"/>
<input type="button" value="取消延迟执行执行代码" onclick="stop()"/><br><br>
<input type="button" value="周期性执行代码" onclick="interval()"/>
