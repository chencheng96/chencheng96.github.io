pyodbc,python

pip install 一般也不用考虑某些库用C语言写的，需要编译的问题：（还有其他安装方法这里就不介绍了，以后有需要再说）

1. 首先，安装python时，一定勾选pip install选项， 以及添加 path。如果这项漏选了可以再次安装选择

2.下载需要的库，这里推荐两个网站

https://pypi.python.org/pypi/pyodbc/4.0.17#downloads

http://www.lfd.uci.edu/~gohlke/pythonlibs/#numpy

上该网站找到需要的库：比如pyodbc-4.0.17-cp36-cp36m-win_amd64.whl 该安装方法最终的安装文件一定是以.whl为后缀的格式


cp36 意即python3.6同理，，，amd64--电脑64位，安情况选择即可如图，cp35 意即python3.5同理，，，amd64--电脑64位，按情况选择即可
下载后注意不要重命名，不然下一步会出错的！！！！！！！！！！

F:\myPython>pip install wheel
Collecting wheel
Downloading wheel-0.29.0-py2.py3-none-any.whl (66kB)
100% |████████████████████████████████| 71kB 45kB/s
Installing collected packages: wheel
Successfully installed wheel-0.29.0

pip install wheel
如果首次用，会安装pip。这个是自动下载安装的。下面是安装后运行pip install wheel 的结果

F:\myPython>pip install wheel
Requirement already satisfied: wheel in d:\programs\python\python36\lib\site-packages

下面是成功安装pyodbc模块的结果

F:\myPython>pip install pyodbc-4.0.17-cp36-cp36m-win_amd64.whl
Processing f:\mypython\pyodbc-4.0.17-cp36-cp36m-win_amd64.whl
Installing collected packages: pyodbc
Successfully installed pyodbc-4.0.17

在python环境中输入 help(‘modules’)   可以列出所有已经安装的模块，如果有pyodbc表示安装成功了。
