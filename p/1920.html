SystemError: error return without exception set


1、基于我们是在pycharm中调试的

在Run->Edit Configurations中添加Environment variable：
PYDEVD_USE_FRAME_EVAL=NO

2、 其他环境可以设置env变量

PYDEVD_USE_FRAME_EVAL=NO
