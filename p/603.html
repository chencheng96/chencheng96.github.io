JavaScript,变量,范围

变量的生命周期又称作用域,指某变量在程序中的有效范围。

全局变量

指从定义变量开始，到整个JavaScript代码结束为止，都可以使用。

在函数定义之外声明的一个变量为全局变量。全局变量作用域是全局的，变量的值在整个持续范围内都可以访问和修改。

局部变量

在函数定义内声明的一个变量。局部变量的作用域是局部的，只在函数内部起作用，每次执行该函数都会创建和销毁该变量，且不能被该函数外的任何事物访问。

如果局部变量和全局变量用相同的变量名，则局部变量将覆盖全局变量

 

JavaScript中没有块级作用域的概念，在其他类C的语言中，由花括号封装的代码块都有自己的作用域。如：

for(var i=1;i<5;i++)
{document.write(i);}
window.alert(i);

 

对于有块级作用域的语音来说，for语句初始化变量的表达式所定义的变量i，只会存在于循环的环境中。而对于JavaScript，由for语句创建的i即使在for循环执行结束后，也依旧会存在于循环外部。

<script type="text/javascript">
function check(m)
{
var i=0;   //定义变量i,变量i的作用范围是整个函数
if(typeof m=="object")
{
var j=5;  //定义变量j变量j的作用范围是整个函数内，而不是if块内。
for(var k=0;k<10;k++) //k的作用范围是整个函数内，而不是循环体内。
{
document.write(k);
}
}
window.alert(k+"\n"+j);     //出循环体，K的值依然存在
}
check(document);
</script>
