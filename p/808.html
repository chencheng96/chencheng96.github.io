Android Studio整理res/Layout中的布局文件(创子目录)

  • 2018-01-08
  • android
  • 公开

第一步：

在res/layout文件夹下创建自己想要的目录（有几个模块就建几个目录）

第二步：

在创建好的模块目录下，创建layout文件夹

第三步：

将布局文件放入相应的layout文件夹

第四步：

打开app/build.gradle，在android{}中加入以下代码

sourceSets {
main {
res.srcDirs =
[
'src/main/res/layout/wangyi',
'src/main/res/layout',
'src/main/res'
]
}
}
