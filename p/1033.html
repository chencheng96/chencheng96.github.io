简介pip安装模块： 1) 使用控制台命令进入到pip的安装路径：D:\Python\Python27\Scripts 2) 使用安装命令行：pip install  PackageName pip安装指定版本命令 cpu版本：pip install tensorflow==1.2.1   (==后面为所要安装的版本号) gpu版本类似：pip install tensorflow-gpu==1.2.1 用pip安装cv2&nb
pip安装模块：
1) 使用控制台命令进入到pip的安装路径：D:\Python\Python27\Scripts

2) 使用安装命令行：pip install  PackageName


pip安装指定版本命令

cpu版本：pip install tensorflow==1.2.1   (==后面为所要安装的版本号)
gpu版本类似：pip install tensorflow-gpu==1.2.1

用pip安装cv2   pip install opencv-python


使用镜像安装
pip install xxxx -i http://mirrors.aliyun.com/pypi/simple/
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple --trusted-host pypi.tuna.tsinghua.edu.cn django

阿里源 pip install -r requirements.txt -i https://mirrors.aliyun.com/pypi/simple/

pip更新模块：
1) 使用控制台命令进入到pip的安装路径：D:\Python\Python27\Scripts
2) 使用更新命令行：pip install -U PackageName

pip卸载模块：
1) 使用控制台命令进入到pip的安装路径：D:\Python\Python27\Scripts
2） 使用卸载命令行：pip uninstall PackageName


删除指定的模块或者包
pip uninstall xxx
