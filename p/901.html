盒模型

一个盒子模型由4部分组成

  1. content
  2. border（边框）指盒子周边的直线，四条边都可以设置边框线。border: 2px solid #f00;（top,bottom,left,right）（width,style,color）
  3. padding（填充）内容与边框线之间的空间。（top,bottom,left,right）
  4. margin（边距）指一个标签与另一个标签之间的间隔。（top,bottom,left,right）

border-style的属性值

  1. none 定义无边框。
  2. hidden 与 "none" 相同。不过应用于表时除外，对于表，hidden 用于解决边框冲突。
  3. dotted 定义点状边框。在大多数浏览器中呈现为实线。
  4. dashed 定义虚线。在大多数浏览器中呈现为实线。
  5. solid 定义实线。
  6. double定义双线。双线的宽度等于 border-width 的值。

页面上的每一个元素都被浏览器看成是一个矩形的盒子，网页就是有许多盒子通过排列堆积而成（上下排列，并列排列，嵌套排列）。在css中分为block box（块级盒子）和inline box（内联盒子），它们分别对应于块级标签和内联标签。

盒子模型的实际width（宽度）和height（高度）：width和height属性设置了内容区域，实际宽度和高度是margin,border,padding,width/height的总和。

块级标签

当做容器，可以容纳内联标签和其它块级标签。主要特点是排他性，在页面布局时独占同一行。标签的宽度（width），高度（height），填充（padding）和边距（margin）都有效。

width和height指的是内容区域的宽度和高度，使用css的长度单位：像素（px），百分比，em（字号的倍数），auto（和父元素宽度相同）

内联标签

只能容纳文本或其他的内联标签。主要特点是允许其他标签与其位于一行，标签的宽度（width）和高度（height）不起作用，其宽度就是自身文字或者图片的宽度。填充（padding）和边距（margin）只有左右方向有效，上下方向无效。

块级标签和内联标签的关系

块级标签 内联标签
独占一行 允许其他标签同处一行
高度，行高及外边距和内边距都可控 高，行高及外边距（上下）和内边距（上下）不可改变，但内外边距（左右）有效。高度可通过line-height设置
宽度默认为100%，除非设定一个宽度 宽度是它的文字或图片的宽度，不可改变
它可以容纳内联标签和其它块标签 只能容纳文本或者其他内联标签

块级标签和内联标签的相互转换

每个标签都有一个display属性，用于确定该属性的类型。比如<div>标签默认display属性值为“block”,成为块级标签，而<span>标签的默认值为“inline”,成为内联标签。如果要实现内联标签向块级标签转换只需要将display属性设置为block就行了。在<a>标签加上"display:block"，转换成块级标签，宽度和高度值生效，并且可以独占一行。

规定应该生成的框的类型，display属性的常用取值

值 描述
none 此元素不会被显示。
block 此元素将显示为块级元素，此元素前后会带有换行符。
inline 默认。此元素会被显示为内联元素，元素前后没有换行符。
inline-block 行内块元素。（CSS2.1 新增的值）
inherit 规定应该从父元素继承 display 属性的值。
  1. 当style.display="block"或style.visibility="visible"时控件或见
  2. 当style.display="none"或style.visibility="hidden"时控件不可见。

不同的是“display”不但隐藏控件，而且被隐藏的控件不再占用显示时占用的位置，而“visibility”隐藏的控件仅仅是将控件设置成不可见了，控件仍然占俱原来的位置。

font标签及其属性

font-family: "lucida grande", "lucida sans unicode", lucida, helvetica, "Hiragino Sans GB", "Microsoft YaHei", "WenQuanYi Micro Hei", sans-serif;

2、font-size 字体大小

css浮动与清除

css浮动（float）可以用在任何元素上，取值：left(左浮动),right(右浮动),none(不浮动，默认值)

清除（clear）

和浮动属性是对立的，去掉某个位置的浮动元素。本质上是迫使它落到浮动元素的下方。属性的取值：（clear:both;）
  • left：左边不允许浮动
  • right：右边不允许浮动元素
  • both：两边都不允许
  • none：默认值，关闭清除属性

定位方式

position定位面板就是把对象放在一个层里来定位，相当于HTML的DIV标签。
position属性取值
  1. absolute：绝对定位，位置不会随网页大小而改变。既然已经绝对地址了，一般就放在页面代码的最下面。 
  2. fixed：特殊的absolute，即fixed总是以body为定位对象的，按照浏览器的窗口进行定位
  3. relative：相对定位，位置会随网页大小改变而改变
  4. static：无定位，默认设置
定位absolute和relative要与top，bottom,right,left结合使用。这几个属性取值方式有三种：auto（自动，这是默认的），长度值（px）和百分比。

overflow溢出属性

当内容超出层的大小，发生溢出时，有四种处理方法：

  1. visible：增加层的大小，从而将层的所有内容显示出来
  2. hidden：保持层的大小不变，将超出层的内容剪裁掉
  3. scroll：总是显示滚动条
  4. auto：只有在内容超出层的边界时才显示滚动条

clip剪辑属性

设置层的可视区域，将需要的部分留下，不需要的部分裁掉。必须将position属性的值设为absolute，此属性方可使用。另外clip裁切的计算坐标都是从左上角（0,0）点开始计算。
clip属性取值方式有两种

  1. auto，默认值，没有剪辑
  2. 依据上，右，下，左的顺序提供自左上角为（0,0）坐标计算的4个偏移数值，其中任一数值都可用auto替换，即此边不剪辑。

对齐方式

垂直对齐（vertical-align）

该属性只对内联标签有效，对块级标签没有效果。

内联标签如<img>,表单中的输入框，单选按钮，文本区域等。

vertical-align属性取值

  • baseline：将对象内容与基线对齐
  • sub：垂直对齐文本下标
  • super：垂直对齐文本上标
  • top：将对象的内容与对象顶端对齐
  • text-top：将对象文本与对象顶端对齐
  • middle：将对象的内容与对象中部对齐
  • bottom：将对象的内容与对象底部对齐
  • text-bottom：将对象文本与对象底端对齐
  • 数值：像素或百分比

文本对齐方式（text-align）

text-align属性的取值

  • left：左对齐
  • center：居中对齐
  • right：右对齐
  • justify：两端对齐

调整单词间距（word-spacing）

该属性可以改变单词与单词之间的距离，对单词本身没有影响。使用单位包括所有度量单位：像素（px），em，百分比等，正负值都可以。

调整字符间距（letter-spacing）

控制字母之间的距离，负值减少，正数增加字母之间的距离。

letter-spacing: 10px;

处理空白区（white-space）

设置处理元素内的空白，white-space属性的取值

  • normal：默认，文本自然换行。如果超过元素宽度，内容自然换到下一行
  • pre：保留换行和空白。只有声明了!DOCTYPE且IE6以上版本才支持
  • nowrap：文本不允许换行，直到遇到标签为止
  • pre-wrap：同pre,但遇到超出范围会自动换行
  • pre-line：同pre，但遇到连续空格会被看做一个空格

伪类（不只用于a标签）

:link 指网页中正常的，还没访问到的，鼠标也没有移到过或点击过的所有链接
:visited 已经点击过的链接
:hover 用户鼠标滑过这个链接时改变链接的展示效果
:active 当用户点击链接时的效果，作用范围是按下鼠标的那短暂的十亿分之一秒


如果改变顺序，hover和active状态将不起作用

a: link{text-decoration: none; }    /*取消链接下划线*/

a: hover{text-decoration: underline; }   /*鼠标经过链接时添加下划线*/
