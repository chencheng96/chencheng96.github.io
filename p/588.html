定位方式

position定位面板就是把对象放在一个层里来定位，相当于HTML的DIV标签。

position属性取值

absolute：绝对定位，位置不会随网页大小而改变

fixed：特殊的absolute，即fixed总是以body为定位对象的，按照浏览器的窗口进行定位

relative：相对定位，位置会随网页大小改变而改变

static：无定位，默认设置


定位位置

absolute和relative要与top，bottom,right,left结合使用。

这几个属性取值方式有三种：auto（自动，这是默认的），长度值（px）和百分比。

<style type="text/css">
#one{
position: absolute;          /*绝对定位*/
top: 50px;          /*距网页顶部50像素*/
left: 50px;          /*距网页左边50px*/
border: 2px #666 solid;       /*边框粗细2像素，实线*/
width: 200px;          /*宽度*/
height: 100px;        /*高度*/
}
</style>
</head>
<body>
<div id="one">元素在网页中的位置由定位方式和定位位置的共同作用。</div>

层

层就是一个容器，用<div>标签来描述。可以使用层的属性来设置层的样式，层的先后顺序由z-index属性确定。

层空间

层空间属性（z-index）就是描述z轴的，属性值越大，层就越位于上面。

<style type="text/css">
#one{
position: absolute;          /*绝对定位*/
top: 30px;          /*距网页顶部50像素*/
left: 50px;          /*距网页左边50px*/
width: 200px;          /*宽度*/
height: 100px;        /*高度*/
background: #000;
z-index: 1;
color: #fff;
}
#two{
position: absolute;          /*绝对定位*/
top: 70px;          /*距网页顶部50像素*/
left: 70px;          /*距网页左边50px*/
width: 200px;          /*宽度*/
height: 100px;        /*高度*/
background: #ccc;
z-index: 2;
}
</style>
</head>
<body>
<div id="one">第一层,z-index属性值为1</div>
<div id="two">第二层,z-index属性值为2</div>

层的宽度和高度

决定层的大小，一般只要设置宽度或高度中一个值，另一个值会主动获取。


层的溢出

overflow溢出属性，当内容超出层的大小，发生溢出时，有四种处理方法：

  • visible：增加层的大小，从而将层的所有内容显示出来
  • hidden：保持层的大小不变，将超出层的内容剪裁掉
  • scroll：总是显示滚动条
  • auto：只有在内容超出层的边界时才显示滚动条

<style type="text/css">
#one{
position: absolute;          /*绝对定位*/
top: 30px;          /*距网页顶部30像素*/
left: 50px;          /*距网页左边50px*/
width: 100px;          /*宽度*/
height: 20px;        /*高度*/
background: #000;
z-index: 1;
color: #fff;
overflow: hidden;       /*溢出内容隐藏*/
}
#two{
position: absolute;          /*绝对定位*/
top: 30px;          /*距网页顶部30像素*/
left: 170px;          /*距网页左边170px*/
width: 100px;          /*宽度*/
height: 20px;        /*高度*/
background: #ccc;
z-index: 2;
overflow: visible;    /*溢出内容可见*/
}
</style>
</head>
<body>
<div id="one">第一层,z-index属性值为1,溢出为隐藏</div>
<div id="two">第二层,z-index属性值为2,溢出为可见</div>

层的剪辑

clip剪辑属性设置层的可视区域，将需要的部分留下，不需要的部分裁掉。

必须将position属性的值设为absolute，此属性方可使用。另外clip裁切的计算坐标都是从左上角（0,0）点开始计算。

clip属性取值方式有两种

  • auto，默认值，没有剪辑
  • 依据上，右，下，左的顺序提供自左上角为（0,0）坐标计算的4个偏移数值，其中任一数值都可用auto替换，即此边不剪辑。

<style type="text/css">
#one{
position: absolute;          /*绝对定位*/
top: 30px;          /*距网页顶部30像素*/
left: 50px;          /*距网页左边50px*/
background: #000;
clip: rect(auto,auto,auto,auto);       /*自动*/
color: #fff;
}
#two{
position: absolute;          /*绝对定位*/
top: 200px;          /*距网页顶部200像素*/
left: 50px;          /*距网页左边50px*/
background: #000;
clip: rect(40px,100px,100px,50px);       /*设置具体的坐标位置*/
color: #fff;
}
</style>
</head>
