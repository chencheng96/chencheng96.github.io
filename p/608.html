JavaScript,Object对象

Object对象实例对于在应用程序中存储和传输数据而言，是很理想的选择。

创建Object对象

var o=new Object();  //构造函数没有指定参数，用于创建一个自定义对象。

var o=new Object(value);   //构造函数制定了参数，可以直接将value参数的值转换为数字对象，布尔对象或字符串对象。

<script type="text/javascript">
var p1=new Object();
p1.name="Tom";
p1.age=20;
document.write(p1.name+"</br>");
document.write(p1.age+"</br>");
var n=new Object(1.23);
var s=new Object("china");
var t=new Object(true);
document.write(n+"</br>");
document.write(s+"</br>");
document.write(t+"</br>");
</script>

还可以在创建对象时，直接给对象的属性赋值。

<script type="text/javascript">
var p={name: "tom",age:20}
document.write(p.name+"</br>");
document.write(p.age);
</script>

constructor属性：返回对象的构造函数

typeof属性可以判断操作数的类型，但如果操作数是对象的话，只会返回“Object”值，可以通过constructor属性引用构造函数来判断对象的类型


<script type="text/javascript">
var p=new Object();
var p2=new Object(12.12);
document.write(p.constructor+"</br>");
document.write(p2.constructor+"</br>");       //  返回一个数字对象
document.write(typeof(p2)+"</br>");           //返回object
</script>

toString()方法：对象的字符串表示

返回一个用于表示对象的字符串，可以输出对象和查看对象的值。

<script type="text/javascript">
var p=new Object();
var p2=new Object(12.12);
document.write(p.toString()+"</br>"); //返回[object Object]
document.write(p2.toString()+"</br>");  //返回内置对象12.12
</script>

复制上述代码在线测试运行效果

toLocaleString()方法：对象的本地字符串表示

返回一个用于表示对象的字符串，只不过该字符串被格式化为适合本地的表示法，日期对象中使用较多。

<script type="text/javascript">
var p=new Date();
document.write(p.toString()+"</br>"); //返回Sat Mar 11 2017 11:28:54 GMT+0800 (中国标准时间)
document.write(p.toLocaleString()+"</br>");  //返回2017/3/11 上午11:28:54
</script>

propertylsEnumerable()方法：判断对象的属性

用来判断某个属性是否为对象本身所拥有的。，而不是继承得到的。

<script type="text/javascript">
var p=new Object();   //定义一个日期对象
p.name="Tom";    //定义对象的属性
document.write(p.propertyIsEnumerable("name")+"</br>"); //返回true
document.write(p.propertyIsEnumerable("age")+"</br>");  //返回false
</script>

isPrototypeOf()方法：判断是否原型对象

判断一个对象是否是另一个对象的原型对象

<script type="text/javascript">
var p=new Object();   //定义一个日期对象
document.write(Object.prototype.isPrototypeOf(p)+"</br>"); //返回true
document.write(Array.prototype.isPrototypeOf(p)+"</br>");  //返回false
</script>

valueOf()方法：返回对象的原始值

返回对象的原始值，如果原始值不存在，则返回对象本身

<script type="text/javascript">
var p=new Object();
var n=new Object(1.23456);
var s=new Object("great");
document.write(p.valueOf()+"</br>");  //返回对象本身
document.write(n.valueOf()+"</br>");
</script>
