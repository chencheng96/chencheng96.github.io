按键精灵连接，读取excel表格，更多可以在全部命令搜索文档进行查询

Call Plugin.Office.OpenXls("F:\taobao\20170507.xls")   //打开excel表格
a2 = Plugin.Office.ReadXls(m,i,q)          //把第m页，i行，q列的单元格数据赋值给a2，a2需要之间进行定义。
Call Plugin.Office.CloseXls()

比如

Dim a2
a2= Plugin.Office.ReadXls(1,2,3)           //把第1页，2行，3列的单元格数据赋值给a2

按键精灵连接，读取access数据库,更多可以在全部命令搜索数据库进行查询

Ret = Plugin.Access.Connect("C:\测试.mdb","密码")    //连接Access数据库，返回的是是否连接成功
Ret = Plugin.Access.SelectData("ACCOUNT", "ID", "id < 200") //查询Access数据库，返回查询结果
Ret = Plugin.Access.Query("Create Table Student(Num int,Name CHAR(10) Not Null,Age int,Weight int)")      //执行SQL语句，返回是否执行成功
Ret = Plugin.Access.Close()    //断开Access数据库连接，返回是否执行成功

date = Plugin.Sys.GetDateTime()//获取系统日期和时间

OSType = Plugin.Sys.GetOSType()//获取系统型号
Select Case OSType
Case 0
MessageBox "windows95/98"
Case 1
MessageBox "windows Me/windowsNT/windowsXP"
Case 2
MessageBox "windows 2003"
Case Else
MessageBox "Unknow"
End Select

Call Plugin.Msg.ShowScrTXT(0, 0, 1024, 100, "屏幕打字", "0000FF")//在屏幕中显示文字
Delay 3000
Call Plugin.Msg.HideScrTXT()

Call Plugin.Media.Beep(880,200)//让主板蜂鸣器报警
