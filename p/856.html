标识符命名规则

Python标识符不能与关键字同名

名的开头和结尾都使用下划线（_）的情况应避免。Python定义的特殊的方法和变量使用的是这种名称。

使用无效标识符会产生SyntaxError异常。

Integral类型

整数

默认十进制，二进制0b，八进制0o，十六进制0x。

数值型操作符：

  • +，-，*，/，//（x除以y舍去小数部分），%（取模，余数），**（x的y次幂） （除*外都有增强版赋值操作符+=...）
  • abs(x)    //返回x的绝对值
  • divmod(x,y)    //二元组形式返回x除以y所得的商和余数（两个整数）
  • pow(x,y)    //x的y次幂，同**
  • pow(x,y,z)    //(x**y)%z的一种写法
  • round(x,n)    //返回浮点数x四舍五入后得到的整数，若给定n，指示返回小数的位数

整数转换函数：

  • bin(i)    //返回整数i的二进制表示（字符串）
  • hex(i)    //返回i的十六进制表示
  • int(x)    //将x转换为整数，失败产生ValueError异常。若x数据类型不知道到整数的转换，产生TypeError异常。若x是浮点数，截取整数部分。
  • int(s, base)    //字符串s转换为整数，失败参数ValueError异常。基数可选范围2~36之间的整数
  • oct(i)    //返回i的八进制表示（字符串）

整数位逻辑操作符：

  • |（OR），^（XOR），&（AND），<<（i左移j位，不带溢出检查），>>（i右移j位，不带溢出检查），~（取反） （除~外都有增强版的赋值操作符|=）

布尔型

True, False

3个逻辑操作符 and, or, not。and 和 or 都使用“短路”逻辑，并返回决定其结果的操作数。not返回True或False。

浮点类型

浮点数（float）

存放双精度浮点数，数值用小数或指数表示（2e-2 即 0.02）。计算机使用基数2表示浮点数，意味着一些十进制数只能大约表示。

需要import sys，这是在我电脑上运行的sys.float_info结果：sys.float_info(max=1.7976931348623157e+308, max_exp=1024, max_10_exp=308, min=2.2250738585072014e-308, min_exp=-1021, min_10_exp=-307, dig=15, mant_dig=53, epsilon=2.220446049250313e-16, radix=2, rounds=1)。sys.float_info.epsilon是机器可以区分出的两个浮点数的最小区别。

  • float.as_integer_ratio()方法可以获取浮点数的小数部分。(3.24).as_integer_ratio()返回(1823957849085051, 562949953421312)
  • float.is_integer()返回浮点数小数部分是否为0。5.09.is_integer()返回False
  • float(8)将整数转换为浮点数。
  • float.hex()将浮点数以16进制表示为字符串。float.hex(3.6)返回 '0x1.ccccccccccccdp+1' 。相反的转换可以用float.fromhex()实现。指数用p表示，因为e是十六进制数字。

math模块（见 comment）提供了更多可用于float的函数，使用前需要import math。

复数（complex）

虚数部分后紧跟一个j 。复数两部分以属性名形式存在，分别为real和imag。    a=1+3j >>> a.real, a.imag输出(1.0, 3.0)

复数数据类型可以作为函数进行调用。complex()接收一个或两个参数，分别是实数和虚数。    >>> complex(3,4.5)返回(3+4.5j)

处理复数使用cmath模块，不能用math模块。（cmath模块大多数是math模块的复数版）

十进制数字（decimal.Decimal）

提供固定的十进制数（完全的准确性），使用前需导入decimal模块。通过 decimal.Decimal()函数创建十进制数。接受整数或字符串（简单十进制数或指数表示）做参数，不能用浮点数。

字符串

使用str数据类型表示，存放Unicode字符序列。str()函数将字符串或者其他可以转换为字符串的数据类型转换为字符串。

可以用单引号，双引号引起。若字符串中包含与所引相同的引号需要用"\"进行转义。

>>> a="my text\""    >>> a     输出'my text"' 

>>> b='my text"'    >>> b    输出'my text"'

Python使用换行作为语句终结，如果在圆括号，方括号，花括号或三引号（"""）包含的字符串内则是例外。

字符串在引号前用r引导，则字符串不进行转义，r"\n"相当于"\\n"

字符串比较

支持通常的比较操作符（<, <=, ==, !=, >, >=）在内存中逐个字节对字符串进行比较。但是不同语言（瑞典，德语……）中字符的排序不同。

字符串分片与步距

序列中单个数据项或字符串中的字符可以用数据项存取操作符[]来提取。'asdfghj'[5] 返回 'h'。字符串索引位置从0开始。

字符串可以使用负索引值。-1代表字符串最后一个字符，-2是倒数第二个，以此类推。 'asdfghj'[-2] 返回 'h'。

字符串分片的3种语法（seq为字符串）：

  • seq[start]    //提取start位置的数据项
  • seq[start: end]    //提取从start开始（含）到end结束（不含）的数据项。若忽略索引值默认起始为0， 终点为len(seq)。    'asdfghj'[0:]返回 'asdfghj'
  • seq[start: end: step]    //每隔step个字符进行提取。若step为负，start默认为-1。step不能为0。    'asdfghj'[::-1] 返回 'jhgfdsa'

字符串操作符与方法

字符串是固定序列，所有可用于固定序列的功能都可用于字符串。包括in（成员关系测试），+=（追加操作），*（复制），*=（增强的赋值复制）

str.format()方法进行字符串格式化

str.format()方法返回一个新字符串，新字符串中，原字符串的替换字段{i}被格式化后的参数替代。

  •  "the {0} the {1}".format(1,"1")     输出     'the 1 the 1'
  • "{who} buy a {thing}".format(thing="apple",who="Tom")    输出    'Tom buy a apple'
  • "{inf[0]} buy a {inf[1]}".format(inf=["Tom","apple"])    输出    'Tom buy a apple'    //字段名可以引用集合数据类型
  • //字段名也可以是字典项
  • import math>>> "pi={0.pi}".format(math)    输出    'pi=3.141592653589793'    //可以存取命名的属性
  • >>> name="li">>> age="23">>> "{name}'s age is {age}".format(**locals())    输出    "li's age is 23"    //当前还在作用范围内的局部变量可以通过内置的locals()函数访问，该函数返回一个字典，键是局部变量名，值是对变量的引用。映射拆分操作符**可以应用于映射（比如字典）产生一个适合传递给函数的键—值列表。

字符串的转换与格式规约

>>> decimal.Decimal("3.4084")
Decimal('3.4084')        #表象形式，提供一个字符串（被Python解释时将重建其表示的对象）
>>> print(decimal.Decimal("3.4084"))
3.4084        #字符串形式，便于阅读

强制提供字符串形式或表象形式，可以通过向字段添加 conversion 指定符（s，用于强制使用字符串形式；r，用于强制使用表象形式；a，强制使用表象形式，仅限于ASCII字符）实现的。

>>> mo="我们"
>>> "{0!s} {0!r} {0!a} {0}".format(mo)
"我们 '我们' '\\u6211\\u4eec' 我们"

格式规约包括：字符串格式规约，整数格式规约，浮点数格式规约

    字符串方法

    方法 描述
    string.capitalize() 把字符串的第一个字符大写
    string.center(width) 返回一个原字符串居中,并使用空格填充至长度 width 的新字符串
    string.count(str, beg=0, end=len(string)) 返回 str 在 string 里面出现的次数，如果 beg 或者 end 指定则返回指定范围内 str 出现的次数
    string.decode(encoding='UTF-8', errors='strict') 以 encoding 指定的编码格式解码 string，如果出错默认报一个 ValueError 的 异 常 ， 除非 errors 指 定 的 是 'ignore' 或 者'replace'
    string.encode(encoding='UTF-8', errors='strict') 以 encoding 指定的编码格式编码 string，如果出错默认报一个ValueError 的异常，除非 errors 指定的是'ignore'或者'replace'
    string.endswith(obj, beg=0, end=len(string)) 检查字符串是否以 obj 结束，如果beg 或者 end 指定则检查指定的范围内是否以 obj 结束，如果是，返回 True,否则返回 False.
    string.expandtabs(tabsize=8) 把字符串 string 中的 tab 符号转为空格，tab 符号默认的空格数是 8。
    string.find(str, beg=0, end=len(string)) 检测 str 是否包含在 string 中，如果 beg 和 end 指定范围，则检查是否包含在指定范围内，如果是返回开始的索引值，否则返回-1
    string.format() 格式化字符串
    string.index(str, beg=0, end=len(string)) 跟find()方法一样，只不过如果str不在 string中会报一个异常.
    string.isalnum() 如果 string 至少有一个字符并且所有字符都是字母或数字则返回 True,否则返回 False
    string.isalpha() 如果 string 至少有一个字符并且所有字符都是字母则返回 True,否则返回 False
    string.isdecimal() 如果 string 只包含十进制数字则返回 True 否则返回 False.
    string.isdigit() 如果 string 只包含数字则返回 True 否则返回 False.
    string.islower() 如果 string 中包含至少一个区分大小写的字符，并且所有这些(区分大小写的)字符都是小写，则返回 True，否则返回 False
    string.isnumeric() 如果 string 中只包含数字字符，则返回 True，否则返回 False
    string.isspace() 如果 string 中只包含空格，则返回 True，否则返回 False.
    string.istitle() 如果 string 是标题化的(见 title())则返回 True，否则返回 False
    string.isupper() 如果 string 中包含至少一个区分大小写的字符，并且所有这些(区分大小写的)字符都是大写，则返回 True，否则返回 False
    string.join(seq) 以 string 作为分隔符，将 seq 中所有的元素(的字符串表示)合并为一个新的字符串
    string.ljust(width) 返回一个原字符串左对齐,并使用空格填充至长度 width 的新字符串
    string.lower() 转换 string 中所有大写字符为小写.
    string.lstrip() 截掉 string 左边的空格
    string.maketrans(intab, outtab]) maketrans() 方法用于创建字符映射的转换表，对于接受两个参数的最简单的调用方式，第一个参数是字符串，表示需要转换的字符，第二个参数也是字符串表示转换的目标。
    max(str) 返回字符串 str 中最大的字母。
    min(str) 返回字符串 str 中最小的字母。
    string.partition(str) 有点像 find()和 split()的结合体,从 str 出现的第一个位置起,把 字 符 串 string 分 成 一 个 3 元 素 的 元 组 (string_pre_str,str,string_post_str),如果 string 中不包含str 则 string_pre_str == string.
    string.replace(str1, str2,  num=string.count(str1)) 把 string 中的 str1 替换成 str2,如果 num 指定，则替换不超过 num 次.
    string.rfind(str, beg=0,end=len(string) ) 类似于 find()函数，不过是从右边开始查找.
    string.rindex( str, beg=0,end=len(string)) 类似于 index()，不过是从右边开始.
    string.rjust(width) 返回一个原字符串右对齐,并使用空格填充至长度 width 的新字符串
    string.rpartition(str) 类似于 partition()函数,不过是从右边开始查找.
    string.rstrip() 删除 string 字符串末尾的空格.
    string.split(str="", num=string.count(str)) 以 str 为分隔符切片 string，如果 num有指定值，则仅分隔 num 个子字符串
    string.splitlines([keepends]) 按照行('\r', '\r\n', \n')分隔，返回一个包含各行作为元素的列表，如果参数 keepends 为 False，不包含换行符，如果为 True，则保留换行符。
    string.startswith(obj, beg=0,end=len(string)) 检查字符串是否是以 obj 开头，是则返回 True，否则返回 False。如果beg 和 end 指定值，则在指定范围内检查.
    string.strip([obj]) 在 string 上执行 lstrip()和 rstrip()
    string.swapcase() 翻转 string 中的大小写
    string.title() 返回"标题化"的 string,就是说所有单词都是以大写开始，其余字母均为小写(见 istitle())
    string.translate(str, del="") 根据 str 给出的表(包含 256 个字符)转换 string 的字符,要过滤掉的字符放到 del 参数中
    string.upper() 转换 string 中的小写字母为大写
    string.zfill(width) 返回长度为 width 的字符串，原字符串 string 右对齐，前面填充0
    string.isdecimal() isdecimal()方法检查字符串是否只包含十进制字符。这种方法只存在于unicode对象。

    \oyy 八进制数，yy代表的字符，例如：\o12代表换行
    \xyy 十六进制数，yy代表的字符，例如：\x0a代表换行
    \other 其它的字符以普通格式输出

    字符串格式规约

    包括填充字符，字段内对齐方式，字段宽度的最大值与最小值。

    格式规约使用冒号（:）引入，其后跟可选的字符对（填充字符，
        对齐字符【<用于左对齐，^用于中间对齐，>用于右对齐】，
        可选的最小宽度（整数）【如果需要指定最大宽度，就在其后使用"." 后跟随一个整数】） 

    >>> s="The sword of truth"
>>> "{0}".format(s)     #default formatting
'The sword of truth'
>>> "{0:>25}".format(s) #right align,minimum width 25
'       The sword of truth'
>>> "{0:-^25}".format(s)        #fill,center align,minimum width 25
'---The sword of truth----'
>>> "{0:.10}".format(s) #maximum width 10
'The sword '

    整数格式规约

    控制填充字符，字段内对齐，符号，最小字段宽度，基数等。

    格式规约使用冒号（:）引入，其后跟可选的字符对（填充字符，
        对齐字符【<用于左对齐，^用于中间对齐，>用于右对齐，=用于在符号与数字之间进行填充】，
        可选的符号字符【+表示必须输出，-表示只输出负号，空格表示为正数时输出空格】，
        可选的最小宽度整数值【可以用字符#引导，以便获取某种基数进制为前缀的输出；也可以以0引导，以便对齐时用0填充】，
        其他进制添加类型字符【b表示二进制，o表示八进制，x表示小写十六进制，X表示大小十六进制，也可以用d表示十进制，c表示输出整数对应的Unicode字符，n表示以场所敏感的方式输出数字】）

    n充分考虑当前的场所，并在产生的输出信息中使用场所特定的十进制字符与分组字符。C场所的十进制字符是一个句点，分组字符是一个空字符串。 

    >>> d=123456
>>> "{0:0=12}".format(d)        #0fill,minimum width 12,填充字符为0，填充位置在符号与数字之间
'000000123456'
>>> "{0:012}".format(d)         #0_pad,minimum width 12,填充字符为0，最小宽度12用0填充
'000000123456'
>>> "{0:*^15}".format(d)        #*fill,center align,min width 15
'****123456*****'
>>> "{0:b} {0:o} {0:x} {0:X}".format(d)        #不同进制（基数）
'11110001001000000 361100 1e240 1E240'
>>>
>>> import locale
>>> locale.setlocale(locale.LC_ALL,"")   #自动确定用户场所
'Chinese (Simplified)_China.936'
>>> x,y=(1234567890,1234.56)
>>> "{0:n} {1:n}".format(x,y)
'1,234,567,890 1,234.56'        #两种不同场所，输出信息有区别
>>> locale.setlocale(locale.LC_ALL,"C")
'C'
>>> "{0:n} {1:n}".format(x,y)
'1234567890 1234.56'

    浮点数格式规约

    可以控制填充字符，字段内对齐，符号，最小字段宽度，十进制小数点后的数字个数，以及是以标准形式，指数形式还是百分数的形式输出数字。

    在整数格式规约的基础上，在可选的最小宽度后面，通过写一个句点并在其后跟随一个整数，指定小数点后数字的个数。结尾处可以加上【e表示使用小写字母的指数形式，E表示使用大写字母的指数形式，f表示标准的浮点形式，g表示通常形式（f或e）】 

    >>> "{:,.4f}".format(3.57821e6+8.23142141e6j)   #复数从Python3.1可以整体处理
'3,578,210.0000+8,231,421.4100j'
>>> "{0.real:.3f}{0.imag:.3f}j".format(3.57821e6+8.23142141e6j)   #复数实部虚部分开处理
'3578210.0008231421.410j'
