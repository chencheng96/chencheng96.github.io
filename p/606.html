JavaScript,构造函数

调用构造函数必须要使用new操作符。构造函数中的参数用于初始化对象，可以在构造函数体内通过this操作符初始化对象的属性和方法。

简单的构造函数

创建对象时，如果没有初始化某个属性，会自动设置为undefined。通过构造函数给对象的属性初始化。创建构造函数方法和创建普通函数的方法相同，使用function语句，然后通过this操作符初始化对象的属性和方法。

有默认值的构造函数

创建对象时，可以设置某属性的默认值，如果没有初始化该属性，就将该属性赋予一个默认值。

<script type="text/javascript">   //创建有默认值的构造函数
function person(name,age,height){
this.name=name;
this.age=age;
if(height==undefined){this.height=180;}
else{this.height=height;}
}
//创建一个对象
var p1=new person("tom",30,173);
document.write("第一个人名字是："+p1.name+"</br>");
document.write("第一个人年龄是："+p1.age+"</br>");
document.write("第一个人身高是："+p1.height+"</br>");
var p2=new person("jarry",35);
document.write("第二个人名字是："+p2.name+"</br>");
document.write("第二个人年龄是："+p2.age+"</br>");
document.write("第二个人身高是："+p2.height+"</br>"); //调用身高的默认属性
</script>

有方法的构造函数

对象除了拥有属性外，还拥有方法，所以在构造函数中也可以定义方法。

<script type="text/javascript">   //创建有默认值的构造函数
function person(name,age,height){
this.name=name;
this.age=age;
this.height=height;
this.sleep=sleep;
}
function sleep(){document.write("这是对象的方法！");} //定义对象的睡眠方法
//创建一个对象
var p1=new person("tom",30,173);
document.write("名字是："+p1.name+"</br>");
document.write("年龄是："+p1.age+"</br>");
document.write("身高是："+p1.height+"</br>");
p1.sleep();
</script>
