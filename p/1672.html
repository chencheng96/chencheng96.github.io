1.在需要更改的组件里新增一个style标签【重点：不加scoped】，然后直接获取class设置样式。注意：为避免污染全局样式，最好不把它放在公共的css里面 ，给父级组件样式添加类名


2.使用  /deep/，深度作用选择器，推荐！！！！
如果你希望 scoped 样式中的一个选择器能够作用得“更深”，例如影响子组件，你可以使用 >>> 操作符：
<style scoped>
.a >>> .b { /* ... */ }
</style>
