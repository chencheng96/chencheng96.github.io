常量

一个固定的值，一但确定就不会被改变，有4种

布尔常量

true和false，一般用于条件语句判断

整数常量

十进制：0~9的数字序列

八进制：第一个数字是0，后面是0~7组成的数字序列

十六进制：以0x或0X开头，后面由0~9和a~f组成的数字序列

浮点常量

整数部分加小数部分表示：12.32,123.12

科学或标准方法表示：5E7,3e5（e后面的整数代表10的指数次幂）

字符串常量

用单引号或双引号括起来的一个或几个字符："javascript",'211','sd12'

如果一个字符串中本身包含单引号和双引号，可以使用字符串转义字符

转义字符 \n \t \b \r \f \\ \' \" \v \0
含义 换行 水平制表符 空格 回车 换页 斜杠 单引号 双引号 垂直制表符 null

变量

常量值不变，变量值可以随程序的执行而发生变化

变量的命名方式

以字母，数字，下划线（_）组成，必须以字母或下划线开头。不能有其他符号。对大小写敏感。

关键字和保留字不能作为变量名。

定义变量

var a;   //定义变量a,没有赋值，此时变量会保存一个特殊的值——undefined

给变量赋值，JavaScript会自动根据赋值数据类型确定变量的类型。

var x=100;  //变量x是数字类型

var y="123";   //y是字符串类型

var xy=true;   //xy是布尔类型

var z=19.5;  //z是数字类型

变量的值

变量可能包含两种不同类型的值：基本类型值和引用类型值。

  • 基本类型值是指保存在栈内存中的简单数据段，这些值完全保存在内存中的一个位置，常见的值有数字类型，布尔类型，字符串类型，空值，未定义值。这5种值在内存中分别占用固定大小空间，我们操作的是他们实际保存的值，所以它们是按值访问的。
  • 引用类型值是指保存在堆内存中的对象，保存的是一个指针。访问这种变量，先从栈内存中读取指针的内存地址，根据这个地址找到保存在堆内存中的值，即引用访问。

引用类型的变量可以添加属性和方法，也可以改变和删除其属性和方法，但是基本类型值不可以。

变量的复制

基本类型的值复制

var a=10;var b=a;//定义变量a,赋值10，定义变量b,把变量a的值赋给b.

<script type="text/javascript">
var a=10;
var b=a;
b=b+20;
window.alert("a的值是"+a+"，b的值是"+b);
</script>

引用类型的值的复制

从一个变量向另一个变量复制引用型的值，也会把栈内存中的值复制一份放到为新变量分配的空间中，但这个值是指针。两个变量实际指向的是同一个对象。

<script type="text/javascript">
var obj1=new Object(); //创建变量对象
var obj2=obj1;
obj1.age=30;
window.alert("变量1的年龄属性值是"+obj1.age+"，变量2的年龄属性值是"+obj2.age);
obj2.name="中国";
window.alert("变量1的姓名属性值是"+obj1.name+"，变量2的姓名属性值是"+obj2.name);
</script>

检测变量的类型

使用typeof操作符，当前数据类型是null，返回的值是object。

<script type="text/javascript">
var str="javascript";
var b=true;
var num=100;
var m;
var n=null;
window.alert("变量str的类型是"+typeof str+"，变量b的类型是"+typeof b+"，变量num的类型是"+typeof num+"，变量m的类型是"+typeof m+"，变量n的类型是"+typeof n);
</script>

如果是引用型的，typeof也会返回object，但可以用专门检测对象的操作符instanceof，

<script type="text/javascript">
var obj=new Object();
window.alert("typeof返回的值"+typeof obj);
window.alert(obj instanceof Object);    //判断变量obj是Object吗？返回true
</script>

常量和变量的区别

常量不能改变，变量的值可以改变。对于每一个变量都有两个值与其关联：

它的数据值，存储在某个内存地址中，有时这个值也称为右值即被读取的值，常量和变量都可以做右值。

它的地址值，保存数据的那块内存的地址，有时也被称为左值，可以认为左值的意思是位置值，常量不能用作左值。

1=3;  //错误，常量1不能做左值
y=3;  //正确
y=y+3;
