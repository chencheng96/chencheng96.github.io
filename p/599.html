算术操作符

加（+）减（-）乘（*）除（/）操作符：二元操作符

模（%）操作符：取余操作符，返回两个操作数的余数

正（+）负（-）号操作符：一元操作符，+无影响，-取反

递增（++）递减（--）操作符：一元操作符,该操作改变操作数的值。

  • 先使用后递增：当操作符在操作数之后时，先使用操作数的值，然后再将操作数递增
  • 先递增后使用：当操作数在操作数之前时，先将操作数的值递增，再使用操作数的值

<script type="text/javascript">
var a=15;
var b=4;
document.write("操作前a,b的值分别为:"+a+","+b+"</br>");
var c1=a+b,c2=a-b,c3=a*b,c4=a/b;
document.write("a+b="+c1+",a-b="+c2+",a*b="+c3+",a/b="+c4+"</br>");
var d=a%b;
document.write("a除以b的余数为:"+d+"</br>");
var e1=+a,e2=-a;
document.write("+a,-a的值分别为:"+e1+","+e2+"</br>");
var f1=a++,f2=++b;
document.write("a++和++b值分别为:"+f1+","+f2+",操作后a,b的值分别为:"+a+","+b+"</br>");
var f3=a--,f4=--b;
document.write("a--和--b值分别为:"+f3+","+f4+",操作后a,b的值分别为:"+a+","+b+"</br>");
</script>

关系操作符

用于对两个操作数的值进行比较，比较大小关系还是相等关系，结果返回一个布尔值true或false。

相等操作符（==）

如果有一个操作数是布尔值，比较之前先转换为数值，false转换成0，true转换成1。

如果一个操作数是字符串，另一个操作数是数值，在比较之前先把字符串转换为数值。

如果一个操作数是undefined，另一个操作数是null，则返回true。

如果一个操作数是对象，另一个不是，则调用对象的valueOf()或toString()方法，将对象转化为原始类型的值再进行比较。

如果操作数类型相同，则比较这两个操作数的值，如果值相等，则返回true，反之返回false。

如果操作数类型的组合与以上组合都不相同，则返回false。

不等操作符（！=）

如果不相等，则返回布尔值true；如果相等，则返回布尔值false。

不等操作符与相等操作符检测的情况正好相反。

全等操作符（===）

相等返回布尔值true，不等返回布尔值false。

只有在两个操作数类型相同，且值也相同的情况下才会返回true。

不全等操作符（！==）

不相等返回布尔值true，相等返回false。

只有在两个操作数类型相同，而且值也相同的情况下才会返回false。

大于，小于操作符（>或<）

满足条件返回true，否则返回false。

如果两操作数都是数字型的，则按照数学方面的规则进行。

如果是字符串型的，按逐个字符进行比较，采用字符在Unicode编号中的数值大小来进行比较。

如果一个是数字型，一个是字符串型，先将字符串型的操作数转换成数字型再进行比较，如果不能转换成数字型，则返回false。

如果操作数不能转换成字符串或者数字，则返回false。

如果一个操作数是NaN则返回false。

大于等于，小于等于操作符（>=或<=）

判断条件及返回结果同上。

字符串操作符

只有一个（+）操作符，作用就是连接两个字符串，产生一个新的字符串。

如果两操作数只有一个字符串，另一个是数字类型的，会自动把数字类型的值转换成字符串类型再做连接。

<script type="text/javascript">
var a="java";
window.alert(a+"script");
</script>

赋值操作符

（=）就是给变量赋值，左值应该是一个变量，右值是一个数值。

逻辑操作符

逻辑与操作符（&&），逻辑或操作符（||）

属于短路操作，如果第一个操作数是false（true），则不再判断第二个操作数。可以用于任何类型的操作数，不只是布尔型。

逻辑非操作符（！）

可以应用于任何值，都返回一个布尔值。（！）先将操作数转换为一个布尔值，然后对其求反。如果操作数值是true，则返回false。

如果操作数是一个对象，返回false。

如果操作数是一个空字符串，返回true。

如果操作数是一个非空字符串，返回false。

如果操作数是数字0，返回true。

如果操作数是非0的数字，返回false。

如果操作数是null，返回true。

如果操作数是NaN，返回true。

如果操作数是undefined，返回true。

<script type="text/javascript">
var a="java";
window.alert(!0+!""+!NaN);          //3，都是true
</script>

位操作符

该操作符先将操作数转换成32为的二进制数值，然后在进行运算，将运算结果转换成十进制数值。

有符号整数，32位中的前31为用于表示整数的值，第32用于表示数值的符号。

按位与操作符（&）

按位或操作符（|）

按位异或操作符（^）

按位非操作符（~）：一元操作符，将操作数所有位取反。本质为：操作数（32位二进制）的负值减1，比如~20结果为-21。

左移操作符（<<）：二元操作符，将第一个操作数（32位）向左移动，移动位数由第二个操作数决定，右补0。比如a=a<<2。

带符号右移操作符（>>）：二元操作符，将第一个操作数（32位）向右移动，移动位数由第二个操作数决定，对于无符号数高位补0。比如a=a>>2.

无符号右移操作符（>>>）：右移时，最左侧的数值都用0来填充。

<script type="text/javascript">
var a="java";
window.alert(~20);
window.alert(-5>>2);
window.alert(-5>>>2);
</script>

特殊操作符

条件操作符（?:）

唯一的一个三元操作符，var max=(a>b)?a:b;   如果条件成立?后的值给max否则:后的值给max

逗号操作符（,）

声明多个变量，var a1,a2,a3=2;

var num1=(5,4,3,2,1); num1的值为1，用于赋值时，逗号操作符总会返回表达式的最后一项。

new操作符

创建对象实例，也可以实例化JavaScript中的内置对象。var d=new Data();  //Data()是内置的日期对象，创建一个名为d的新日期对象然后调用这个日期对象的有关方法和属性。

void操作符

一元操作符，可以作用在任何类型的操作数之前，可以让操作数进行运算，但却舍弃运行之后的结果。

<script type="text/javascript">
var a=10;
var b=20;
window.alert("a+b的值为："+(a+b)+",void(a+b)的值为："+void(a+b));
</script>

typeof操作符

一元操作符，作用在操作数之前，返回一个字符串，说明操作数的类型。window.alert(typeof 10);

对象属性存取操作符（.）

二元操作符，第一个操作数是对象实例，第二个操作数是属性或方法：

<script type="text/javascript">

delete操作符

用来删除变量，对象的属性，数组中的元素，如果删除成功返回true，删除失败返回false。

in操作符

二元操作符，要求第一个操作数是字符串类型或可以转换为字符串类型的其他类型，第二个操作数是数组或对象。只有第一个操作数的值是第二个操作数的属性名才会返回true，否则返回false。

<script type="text/javascript">
var a=["java","scrip"];
var b="java";
window.alert(b in a);
</script>

instanceof操作符

用来判断对象与对象实例之间的关系，是二元操作符，要求第一个操作数是对象或数组的名字，第二个操作数是对象类的名称。如果第一个操作数是第二个操作数的实例则返回true，否则返回false。

<script type="text/javascript">
var a=["java","scrip"];
window.alert(a instanceof Array);
window.alert(a instanceof Number);
</script>
