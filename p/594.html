在HTML中使用JavaScript

直接在页面中嵌入JavaScript代码

<script type="text/javascript">

function hello{

window.alert("hello world!");

}

</script>

包含外部JavaScript代码

hello.js文件：

function hello{

window.alert("hello world!");

}

然后进行调用


<script type="text/javascript" src="hello.js"/>

也可以指定外部的某个链接

<script type="text/javascript" src="http://www.iyaka.cn/hello.js"/>


JavaScript语法结构


  • 大小写敏感
  • 自动忽略程序中的空格和换行符
  • 每一行语句结束时通常都有分号（;）
  • "//"用于单行注释，"/**/用于注释多行。
