JavaScript,函数

函数用来把程序组织成小的，独立的单元。每个函数封装一个或一组算法，这些算法分别应用在特点的数据集上。

定义函数

JavaScript中使用function语句定义函数。

function 函数名(参数1,,参数2,参数3,...){代码块; return 返回值;}

调用函数

直接调用没有返回值的函数

<script type="text/javascript">
function hello(str) //定义一个函数，函数名为hello
{window.alert(str);   //没有return语句，此函数没有返回值
}
hello("JavaScript!");//直接调用没有返回值的函数
hello("hello_the_word");//直接调用没有返回值的函数
</script>

函数的参数

JavaScript不介意传递进来多少个函数，也不在乎传进来的参数是什么数据类型。即使定义两个参数，调用时可以只传递1个或者3个参数。如果传递的参数个数小于函数定义的参数个数，会自动将多余的参数值设为undefined，如果多于函数定义的参数，那么多余的参数会被忽略掉。

JavaScript中的参数在内部用一个数值表示，在函数体内可以通过arguments对象来访问这个参数数组，从而获取传递给函数的每一个参数。可以通过arguments[0]来访问第一个元素，通过arguments[1]访问第二个元素...使用length属性来确定传递进来多少个参数。

<script type="text/javascript">
function sum(x,y)
{
document.write("第一个传递进来的参数值是："+arguments[0]+"</br>");
document.write("第二个传递进来的参数值是："+arguments[1]+"</br>");
document.write("一共传递的参数个数是："+arguments.length+"个");
}
sum(3,5);
</script>

系统函数

系统内置的函数

编码函数escape()

将字符串中的非文字，数字中的字符转换成相应的ASCII码，将传入参数中的所有空格，标点，符号，重音字符以及其他任何非ASCII字符替换为%xx的编码形式，其中xx与其所表示的字符的十六进制数表示形式相同。如空格字符的十六进制表示形式为0x20，则此时xx应为20，即escape(" ")返回"%20"。

解码函数unescape()

与escape()的作用相反，将ASCII码的文字转换成一般文字。

<script type="text/javascript">
var str="!@#$"
document.write("编码前的字符串："+str+"</br>");
document.write("编码后的字符串："+escape(str)+"</br>");
str=escape(str);str=unescape(str);
document.write("解码后的字符串："+str);
</script>

求值函数eval()

把一个字符串当作一个JavaScript表达式去执行它。

<script type="text/javascript">
var str="2+3";
var num=eval("2+3"); //把表达式执行算术运算
document.write("str的值："+str+"</br>");         //输出字符串
document.write("num的值："+num+"</br>"); 
</script>

数值判断函数isNaN()

判断某个变量是否是非数字值，如果是，返回true，否则返回false。

isNaN(123); //false

isNaN("great"); //true

整数转换函数parseInt()

把字符串转换成整数。

parseInt(字符串,基数);

第一个参数字符串是必须的。第二个参数“基数”是可选的，表示要解析的数字的基数，该值介于2~36，若省略或该参数为0时，数字以10进制为基数解析；若以“0x”或“0X”开头，将以十六进制为基数；若该值小于2或大于36，则parseInt()将返回NaN。

<script type="text/javascript">
parseInt("10");     //省略基数，默认为10进制，得到10
parseInt("10",10);     //10进制，得到10
parseInt("10",2);     //基数为2，得到十进制数2
parseInt("10",8);     //基数为8，得到十进制数8
parseInt("10",16);     //基数为16，得到十进制数16
parseInt("2A5C",16);     //基数为16，得到十进制数10844
</script>

浮点数转换函数parseFloat()

把字符串转换成浮点数据

parseFloat(字符串);

如果字符串的第一个字符不能被转换为数字，那么parseFloat()会返回NaN。

<script type="text/javascript">
document.write(parseFloat("16"));  //得到16
document.write(parseFloat("16.00"));  //得到16
document.write(parseFloat("16.98"));  //得到16.98
document.write(parseFloat("16 12 32"));  //得到16，后面的省略了
document.write(parseFloat(" 16"));  //得到16，前面的空格不受影响
document.write(parseFloat("he 16"));  //得到NaN，第一个字符是字母，全部省略
document.write(parseFloat("16 asdqwe"));  //得到16，后面字符串省略了
</script>
